# 项目初筛：用投资标准快速判断要不要深看

这是 Anthropic 官方的 PE 项目初筛 skill。收到一份 CIM / teaser 后，用你基金的投资标准做快速 pass/fail：

- 抽取交易关键指标（规模、行业、财务、股权结构）

- 对照基金投资标准跑一遍打分框架

- 输出一页纸的初筛 memo：推荐 / 不推荐 + 理由

适合：deal flow 太多看不过来的投资人。先用 Claude 过一遍，把时间留给真正值得深看的项目。
