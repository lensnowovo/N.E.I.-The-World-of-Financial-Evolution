# 合成生物 / 生物制造投资研究框架

## 这是什么
合成生物学（生物制造/工业生物技术）的投资分析框架。这是横跨"制造业"和"生物科技"的跨界赛道，**核心矛盾是：技术想象力巨大但商业化兑现难**——资本在 2021 年疯狂涌入（百亿级），但 2023-2024 年规模化收入的公司寥寥。2024-2026 的认知迭代：**从"讲故事"到"看产品"**——技术故事不再万能，产品做出来、卖出去才融得到钱。适合 PE/VC 早期组、产业资本（招商局/国投/国药/中粮）、地方政府引导基金。

## 行业研究框架（6 维度）

### 维度 1：产业链全景与竞争格局

```
上游（使能技术 / 工具层）     ← 平台型公司
├── DNA 读/写/改（基因编辑/合成）
├── 底盘菌库 / 工程酶库
├── 高通量自动化平台
└── AI + 合成生物（AI4S）

中游（菌株设计 + 发酵工艺）  ← 核心技术层
├── 菌株理性设计 / 代谢通路优化
├── 发酵工艺 / 分离纯化
└── 中试放大

下游（产品 + 应用）           ← 商业化层
├── 高附加值新分子（重组胶原蛋白/HMO/角鲨烷/麦角硫因）
├── 大宗化学品（丁二酸/聚酰胺/PLA/乙醇）
├── 医药中间体
├── 农业饲料 / 食品添加剂
└── 生物基材料
```

**全球格局**：美国先驱（Zymergen/Amyris/Ginkgo）接连折戟；中国对标切换为帝斯曼/巴斯夫/杜邦。全球化工品 23 亿吨/年，生物基仅 1.8 亿吨——替代空间巨大但路漫漫。

### 维度 2：选品策略（决定生死的战略决策）

> **"选品决定生死"**——这是合成生物投资的共识。

**两种路径深度对比**：

| 维度 | 赌"新"分子 | 卷"旧"分子（大宗） |
|---|---|---|
| **典型产品** | HMO/角鲨烷/麦角硫因/重组胶原蛋白 | 丁二酸/聚酰胺/纤维素乙醇/PLA |
| **代表成功案例** | 巨子生物、华熙生物、锦波生物 | 凯赛生物、华恒生物、首钢朗泽 |
| **附加值** | 高（公斤级 → 万元级） | 低（吨级 → 万元/吨） |
| **市场天花板** | 单品有限，需爆款 | 单品可成就一家上市公司 |
| **重资产程度** | 相对轻 | 极重（需建大厂） |
| **监管门槛** | 高（食品/医用需双重审批） | 低（工业品标准） |
| **核心风险** | 下游需求未爆、价格已卷 | 与巨头拼成本、周期性价格波动 |
| **典型失败** | 麦角硫因（15 万→3 万/公斤，需求未起） | Amyris（生物燃料，油价下跌致财务危机）、杜邦（纤维素乙醇工厂抛售） |

**峰瑞资本的"平衡点"原则**：5000-10000 吨规模 + 较高附加值 + 不错市场空间——不追求超高附加值的新分子，也不追求几十万吨大宗。

**"命题作文"原则**：市场需求明确或有渠道拿到订单时，才投入量产。

### 维度 3：商业化壁垒——"不像创新药，不像半导体"

> **合成生物的壁垒困境**：菌株设计和量产工艺，**不似创新药专利保护那样成熟，也不像半导体等高端制造壁垒显著**。"生物技术有时像一层窗户纸，有了方向后，别人花时间钻研也可能做得出来。"

**壁垒构建策略**：
1. **在技术领先的时间差里，建立规模和渠道先发优势**——而非纯靠专利。
2. **菌株专利 + 工艺 know-how 组合**——双重保护。
3. **监管准入（食品/医用）**——双重审批（"添加成分 + 公司特定菌株/工艺"）= 准入壁垒。批文"申报材料堆起来有 2 米多高"。
4. **规模效应**——单位成本随产能下降。

### 维度 4：商业化兑现的真实挑战

**真实世界的"坑"**（来自一线创业者）：
1. **ESG 概念撬动的销售有限**——客户最终只关心降本增收，不为绿色概念溢价买单。
2. **伪客户多、真订单少**——口头意向 ≠ 付费订单。
3. **原料成本测算外**——如秸秆看似废料，收购/运输/加工都有成本（"农民往里加石子增重"）。
4. **生物基 vs 石油基的定价逻辑**：
   - 价格持平或高 10-30%：可能全面替代。
   - 高 2-3 倍：小范围高端市场。
   - 高 3 倍以上：概念性商品，无市场。

**价格卷的残酷现实**：
- 麦角硫因：15 万→3 万/公斤（1 年内）。
- 依克多因：~3000 元/公斤，仍在下探。
- PLA（聚乳酸）：降至 2 万多/吨，贴近成本线。
- NMN：被禁售。

### 维度 5：中国特色——国家队入场与资本结构变化

**2024-2026 政策定调**：
- "生物制造"被定义为战略性新兴行业（中央经济工作会议）。
- 与商业航天、低空经济并列"新质生产力"新增长引擎。

**四类资本玩家**：
| 类型 | 关注点 | 估值容忍度 |
|---|---|---|
| **国央企基金**（招商局/国投/国药/中粮） | 战略价值、平台型基础设施 | 高 |
| **地方政府引导基金** | 招商落地（搬工厂换投资） | 较高（但有上限 ~1 亿/项目 + 终身追责） |
| **市场化财务 VC** | 估值与商业化进程匹配 | 低（参考上市公司 PE/PS） |
| **产业方 CVC**（如华恒生物） | 资源换估值（产能/渠道） | 中 |

**现实约束**：地方基金单项目上限 ~1 亿元，"不足以救命"；市场化基金处于募资低谷，给估值低于企业预期。

### 维度 6：BD 交易与出海——新趋势

**对标创新药的 BD 模式**：合成生物正在探索类似管线 License-out 的交易——"首付款 + 合作研发 + 销售分成"。

**合作开发模式兴起**：
- 元素驱动（西湖大学张科春）+ 牧原股份 → 牧原安粮（氨基酸/豆粕减量）。
- 摩珈生物 + 湖南海利（技术转让/合作开发）。
- 凯赛生物 + 招商局集团（66 亿定增 + 2023-2025 采购 1/8/20 万吨聚酰胺）。

**出海必要性**：国内监管未准入的产品（阿洛酮糖/甜菊糖苷），先走海外出口（GRAS 认证）。

## 壁垒判断

### 真壁垒
1. **监管准入（食品/医用的双重审批）**——批文稀缺，申报材料 2 米高。
2. **菌株专利 + 工艺 know-how 组合**——虽不如创新药成熟，但仍构成时间差。
3. **规模效应 + 成本曲线领先**——首钢朗泽/凯赛生物级别。
4. **下游渠道 + 客户绑定**——真实订单 + 复购。
5. **平台型技术（基因编辑/酶库/菌库）**——可延伸多管线。

### 假壁垒
1. **"技术领先"无规模支撑**——"窗户纸"被捅破只是时间问题。
2. **ESG/绿色概念**——客户不为概念买单。
3. **"顶刊论文 = 商业价值"**——学术 ≠ 商业。
4. **单一分子依赖**——麦角硫因价格崩盘即业绩崩盘。

## 红旗识别
1. **纯研发无收入 + 估值 > 10 亿**——PS 倍数高企，"打折卖也接不起"。
2. **单一分子贡献 > 70% 营收**——价格波动风险。
3. **生物基价格 > 石油基 3 倍**——概念性商品，无真实市场。
4. **客户口头意向多但付费订单少**——伪客户陷阱。
5. **宣称"对标 SpaceX/Ginkgo"但无规模化收入**——叙事与现实脱节。
6. **单位成本未随规模下降**——工艺未真正成熟。
7. **批次稳定性 < 80%**——量产风险。
8. **重资产投产但产能利用率 < 40%**——固定成本拖累。
9. **核心科学家兼职 + 关键期缺位**——创业决心不足。
10. **融资主要靠地方招商基金（< 1 亿）**——"不足以救命"。
11. **管线/应用场景"藏起来"但实际是没找到**——方向不清。
12. **下游监管批文申请停滞**——准入风险（食品/医用致命）。
