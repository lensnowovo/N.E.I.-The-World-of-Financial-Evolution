# Teaser 匿名推介：一页纸勾起买家兴趣（不泄公司名）

这是 Anthropic 官方的 Teaser（匿名推介） skill。卖项目第一阶段发的那页不具名材料：

- 匿名描述：行业、规模、增长、地理，但不露公司名

- 一页纸突出 3-5 个核心卖点

- 引导买家签 NDA 才给完整 CIM

- 语调克制专业，避免过度承诺

适合：FA 做项目早期触达买家。Teaser 决定能拉到多少买家签 NDA 进下一轮。
