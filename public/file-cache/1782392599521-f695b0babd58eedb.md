# 专利检索与分析方法论 · Skill 草稿

## 这是什么

面向 PE/VC/FA 投研人员的**专利检索、布局分析与 FTO（自由实施）评估 SOP**。硬科技投资中，专利是最核心的 IP 尽调项——技术是否自由实施、是否侵犯他人专利、核心专利何时到期、竞争对手专利布局、技术空白区在哪，这些问题的答案都藏在专利数据库里。但很多投研人查专利只会用 Google Pat Patent 搜公司名，不会用 IPC 分类、不会查同族、不会做 FTO、不会判断权利要求保护范围。本 Skill 把专利分析拆成"选平台 → 检索 → 法律状态 → 同族引证 → 布局分析 → FTO 评估"六步，可作为任何硬科技/生物医药/半导体/新材料尽调的前置动作。适合投硬科技/生物医药/半导体/新材料/新能源的投资人、IP 尽调人员、技术战略。

## 方法论框架（6 维度）

### 维度 1：检索平台选型（去哪查）

| 平台类型 | 代表平台 | 特点 | 适用场景 |
|---|---|---|---|
| **官方免费** | 中国专利查询系统 cpquery.sipo.gov.cn / 专利检索及分析系统 pss-system.cnipa.gov.cn / USPTO / EPO Espacenet | 权威、免费、覆盖官方数据 | 法律状态查询、基础检索 |
| **商业专业** | 智慧芽 PatSnap / 大为 INNOJOY / Incopat / Patsnap Eureka | 专利地图、技术趋势、竞争分析 | 企业级战略布局、深度分析 |
| **学术工具** | Google Patents / Lens.org | 简单快捷、覆盖广 | 初步筛查 |
| **特定领域** | PubMed（医药）/ CAS（化学） | 领域索引 | 生物医药/化学深挖 |

**选择建议**：
- **个人/小团队基础需求**：中国专利网 + Google Patents（免费）。
- **企业级战略布局**：智慧芽 / 大为（付费，年研发投入 > 500 万建议上）。
- **跨境技术调研**：EPO Espacenet（多语言 + 同族追踪）。

### 维度 2：检索方式（怎么查）

#### 三类主流检索方式

##### 1. 导航检索（IPC 分类逐级筛选）

> 适合"模糊找方向"——不知道具体技术名，但有领域概念。

**IPC 分类示例**：
- `H01M` → 电池
- `H01M 10` → 二次电池（可充电）
- `H01M 10/0525` → 锂离子电池

**技巧**：先定位大类 → 细分到子类 → 加入关键词联合检索。

##### 2. 高级检索（关键词 + 逻辑运算符）

> 适合"精准找特定技术"。

**检索式示例**：
```
(锂电池 AND 正极材料) NOT 磷酸铁锂
(半导体 AND 光刻) AND (ArF OR EUV)
(CAR-T AND 实体瘤) AND (中国 OR 美国)
```

**运算符**：AND（与）/ OR（或）/ NOT（非）/ `*`（通配符）/ `""`（精确匹配）。

##### 3. 法律状态检索（查有效/失效）

> 适合"判断专利是否仍有效"——避免为过期专利付许可费。

输入：专利号 / 公开号 / 申请号。
输出：有效 / 失效 / 驳回 / 待审 / 实质审查中。

**投研应用**：查某公司宣称的"核心专利"是否仍有效——若已失效，技术壁垒存疑。

### 维度 3：核心分析维度（看什么）

#### 检索后重点关注的 4 大指标

| 指标 | 含义 | 投研判断 |
|---|---|---|
| **技术活跃度** | 近 5 年该领域专利申请量增速 | 增速 > 20% = 高增长领域 |
| **竞争集中度** | 前 10 名申请人专利占比 | > 60% = 垄断风险高，新进入者难 |
| **技术生命周期** | 申请-授权曲线判断 | 导入期/成长期/成熟期/衰退期 |
| **核心专利被引次数** | TOP10 高被引专利 | 被引 > 100 = 基础性突破 |

#### 标准专利分析报告 5 要素

1. **技术领域概述**：该领域总体专利申请趋势。
2. **全球/中国分布**：地域布局、主要市场。
3. **主要申请人分布**：竞争格局（企业/高校/研究机构）。
4. **核心专利解读**：被引 TOP10 专利的技术要点。
5. **技术空白区建议**：哪些子领域尚无密集布局 → 创新机会。

### 维度 4：同族专利与引证分析（深挖）

#### 同族专利（Patent Family）

> 同一发明在多国申请的专利集合——是判断技术全球布局和申请人意图的关键。

**应用场景**：
- **判断技术重要性**：申请国越多 = 申请人越重视 = 商业价值越高。
- **跨境诉讼证据**：同族在多国有效 = 全球维权基础。
- **判断申请人市场意图**：只在中国申请 = 只做中国市场；中美欧三地申请 = 全球化意图。

**查询方式**：Espacenet / 智慧芽的"同族信息"栏目。

#### 引证分析（Citation Analysis）

| 类型 | 含义 | 投研判断 |
|---|---|---|
| **后向引证（引用别人）** | 该专利引用了哪些在先专利 | 技术基础、依赖关系 |
| **前向引证（被别人引用）** | 该专利被后续专利引用 | 技术影响力——被引 > 100 = 基础性突破 |
| **自引比例** | 引用自己其他专利的比例 | 高自引 = 持续布局，技术体系化 |

**投研应用**：找被引 > 100 的专利 → 该领域的里程碑式技术 → 查谁持有 → 判断竞争壁垒。

### 维度 5：FTO（自由实施）评估

> FTO（Freedom to Operate）是投资硬科技最重要的 IP 尽调——判断目标公司的技术是否侵犯他人有效专利。

#### FTO 评估步骤

1. **拆解技术特征**：把目标公司的产品/技术拆成若干技术特征。
2. **检索相关有效专利**：按技术特征检索，限定"有效"和目标市场。
3. **权利要求比对**：把目标专利的独立权利要求与目标公司技术逐项比对。
4. **侵权判断**：全面覆盖原则 + 等同原则——若目标技术落入权利要求保护范围，构成侵权。
5. **风险评估**：
   - 高风险：核心技术落入他人有效专利 → 投资否决项。
   - 中风险：外围技术可能侵权 → 要求回购/赔偿条款。
   - 低风险：无有效专利覆盖 → FTO 清晰。

**投研场景**：FTO 是硬科技投资的核心尽调项——一个技术再先进，若侵犯他人有效专利，商业化即被阻断。

#### 投资条款中的 IP 保护

- **IP 陈述与保证**：创始人承诺不侵犯第三方 IP。
- **IP 侵权补偿条款**：若发生侵权诉讼，创始人承担赔偿。
- **回购条款**：若核心 IP 被判侵权，投资人有权要求回购。

### 维度 6：专利布局策略分析（看懂壁垒）

#### 企业专利布局的 5 种模式

| 模式 | 含义 | 壁垒强度 |
|---|---|---|
| **伞形布局** | 核心专利 + 大量外围专利，形成保护伞 | 最强（如华为 5G） |
| **网络布局** | 多个技术方向交叉申请，形成网络 | 强 |
| **单点突破** | 只有一两个核心专利，无外围 | 弱（易绕过） |
| **跟进布局** | 围绕他人核心专利申请改进专利 | 中（防御性） |
| **阻断布局** | 在竞争对手技术路径上申请专利，阻断对手 | 中（进攻性） |

**投研应用**：查目标公司专利布局模式——单点突破 + 无外围 = 壁垒弱；伞形布局 = 真壁垒。

#### 核心专利到期追踪

> 专利保护期：中国发明专利 20 年（从申请日起算）。核心专利到期 = 技术壁垒消失 = 仿制/竞争进入。

**投研应用**：
- 生物医药：查核心化合物专利到期日 → 判断仿制药进入时点。
- 半导体：查核心工艺专利到期 → 判断技术扩散。
- 查目标公司"宣称的核心专利"实际剩余保护期——若剩余 < 5 年，壁垒即将消失。

## 行业核心指标

| 指标 | 健康阈值 | 说明 |
|---|---|---|
| **发明专利占比** | > 60% | 专利质量（实用新型/外观 ≠ 技术壁垒） |
| **核心专利剩余保护期** | > 10 年 | 壁垒持续性 |
| **同族国家数** | ≥ 3 国 | 全球化意图与技术重要性 |
| **前向被引次数（核心专利）** | > 50/100 | 技术影响力 |
| **专利布局模式** | 伞形/网络 | 真壁垒 vs 单点 |
| **技术空白区** | 有可布局空间 | 创新机会 |
| **FTO 清晰度** | 无有效专利覆盖核心路径 | 商业化可行性 |
| **近 3 年专利增速** | > 20% | 持续创新能力 |
| **专利与论文一致性** | 核心论文已申请专利 | 学术突破已 IP 化 |

## 红旗识别
1. **只会搜公司名，不会用 IPC/关键词**——漏掉大量专利。
2. **不区分发明/实用新型/外观**——实用新型和外观不是技术壁垒。
3. **不查法律状态**——"核心专利"可能已失效。
4. **不做 FTO 评估**——硬科技投资最大盲区，核心技术可能侵犯他人。
5. **只看专利数量，不看质量**——100 个外围专利不如 1 个被引 100 次的核心专利。
6. **不查同族**——漏掉全球布局信息，误判技术重要性。
7. **不看权利要求书**——只看标题/摘要，误判保护范围。
8. **不追踪核心专利到期日**——壁垒即将消失却不知情。
9. **不分析布局模式**——单点突破误判为强壁垒。
10. **专利与论文脱节**——PI 论文惊艳但未申请专利 = 无 IP 保护。
11. **自引比例过高（> 70%）**——可能是刷数，技术影响力存疑。
12. **专利申请集中于 1 年**——突击申请，非持续创新。
13. **发明专利占比 < 40%**——IP 质量低。
14. **核心专利剩余保护期 < 5 年**——壁垒即将消失。
