# 生物医药 / 创新药 / CGT 投资研究框架

## 这是什么
创新药 / 生物药 / 细胞与基因治疗（CGT）的投资分析框架。这是 PE/VC 医疗组的**独立语言体系**——完全不同于硬件/SaaS/消费。核心特征：**超长周期（10-15 年）、超高失败率（I 期到上市仅 9.6%）、监管驱动（IND/NDA/BLA）、专利悬崖**。适合做创新药管线尽调、估值、立项。**这是上一轮《05 医疗服务/器械》未覆盖的最大缺口。**

## 行业研究框架（6 维度）

### 维度 1：药物研发全生命周期与监管节点

```
药物发现 → 临床前 → IND申报 → Ⅰ期 → Ⅱ期 → Ⅲ期 → NDA/BLA → 上市 → 专利到期
  3-5年    3-6年      ↓       1-2年   2-3年   2-3年   ↓     ↓       ↓
                    安全性   剂量探索 有效性 大样本  审批  商业化  仿制药
                    (20-100人)(100-300)(300-3000)          放量    冲击
                                                                    
总周期：10-15 年，成本：10-20 亿美元/款
```

**关键监管节点**：
- **IND（Investigational New Drug）**：临床许可，从临床前进入人体试验。
- **NDA（新药申请）/ BLA（生物制品许可申请）**：上市许可。
- **FDA Breakthrough / Fast Track / Orphan Drug**：加速审批通道。
- **CDE（中国药监）**：中国版突破性治疗/附条件批准/优先审评。

### 维度 2：临床各阶段成功率（rNPV 风险调整的基础）

> 估值的核心是"概率调整"——不同阶段给不同的成功概率折扣。

| 阶段 | 成功率（美国数据） | 累计成功率（从该阶段到上市） | 投资含义 |
|---|---|---|---|
| 临床前 → IND | ~60% | ~5-10% | 极早期，天使/孵化 |
| Ⅰ期 → Ⅱ期 | ~60% | ~10-15% | 早期 VC |
| Ⅱ期 → Ⅲ期 | ~30-50%（最大淘汰点） | ~15-25% | **最关键分水岭** |
| Ⅲ期 → NDA | ~60-85% | ~50-70% | 成长期 PE / IPO |
| NDA → 上市 | ~90% | ~90% | 接近确定性 |

**关键认知**：**Ⅱ期到 Ⅲ 期是最大的淘汰点（50%+ 项目死于此）**。这也是为什么 Ⅱ 期数据读出（readout）是创新药投资最重要的催化剂——数据好 = 估值翻倍，数据差 = 腰斩甚至清零。

### 维度 3：管线估值（rNPV 法）

**rNPV = 风险调整净现值（Risk-adjusted Net Present Value）**——美国 Biotech 公司的标准估值法。

**5 步计算流程**：
1. **构建 TPP（目标产品概要）+ CDP（临床开发计划）**：定义适应症、终点、给药方案、上市时间。
2. **预测销售峰值（Peak Sales）**：患者数 × 渗透率 × 定价 × 用药时长。
   - 患者数 = 新增/累计病例 × 就诊率 × 治疗率 × 药物使用率。
3. **预测全生命周期现金流**：上市后 5-7 年达峰 → 稳定 → 专利到期（10-15 年后）→ 仿制药冲击 → 销售下滑。总生命周期约 20 年。
4. **风险调整**：按当前临床阶段乘以成功概率（见维度 2）。
5. **折现**：通常 10-12% 折现率（高于普通 DCF，因研发风险已单独调整，折现率反映资金时间价值）。

**简化法（销售峰值 PS）**：
> 在 45-55% 单产品利润率下（85-95% 毛利 - 35-45% 销售费用），**单产品生命周期利润现值 ≈ 销售峰值 × 3-4 倍 PS**。

**管线组合估值**：公司估值 = Σ 各管线 rNPV + 平台价值 + 现金。

### 维度 5：中国特色——工程师红利与 BD 出海

**"DeepSeek 时刻"**（2025《华尔街日报》评述）：中国创新药产业从"跟随式创新"转向"全球新分子主产地"。

**三大中国特色优势**：
1. **全产业链工程师红利**：Biotech + CRO（研发外包）+ CDMO（生产外包）+ 医院——效率全球最高、成本最低。CDMO 海外收入占比 > 80%。
2. **BD 出海爆发**：在美国药企交易中，中国分子占比从近乎零升至 26.4%（2025H1）。海外授权首付款 + 里程款已成 Biotech 重要现金流。
3. **政策红利**：2024《全链条支持创新药发展实施方案》+ 2025 医保丙类目录（创新药支付突破口）+ "十五五"首提"支持创新药"。

### 维度 6：Pharma vs Biotech 两种投资逻辑

| 维度 | Pharma（大药企） | Biotech（生物科技） |
|---|---|---|
| **商业模式** | 仿制药/成熟药现金流 + 自研/并购管线 | 纯研发驱动，无/少收入 |
| **估值基础** | PE/PEG + 管线溢价 | **rNPV（核心）+ 平台价值** |
| **风险特征** | 低（分散管线） | 高（单一管线失败 = 腰斩） |
| **投资逻辑** | "时间的朋友"——资金积累 + 销售能力 + 并购 | 重磅产品爆发力 + 出海授权 |
| **核心能力** | 资金 + 销售体系 | 产品力 + 临床设计 |
| **对标** | 恒瑞/中国生物制药/信达 | 科伦博泰/百利天恒/康方 |

## 行业核心指标

| 指标 | 健康阈值 | 说明 |
|---|---|---|
| **管线阶段分布** | 有 ≥ 1 个 Ⅱ/Ⅲ 期 | 梯度合理性 |
| **临床数据读出节点** | 12-18 月内有催化剂 | 估值驱动事件 |
| **研发费用率** | > 30%（Biotech）/ 10-20%（Pharma） | |
| **现金 runway** | > 24 个月 | Biotech 生命线 |
| **BD 授权交易（首付款）** | > 1000 万美元 | 出海能力信号 |
| **CDE 突破性/附条件批准** | 有 | 监管认可 |
| **销售峰值预测** | > 5 亿美元 | 重磅药物门槛 |
| **毛利率（上市产品）** | > 85% | |
| **专利剩余年限（核心产品）** | > 10 年 | |
| **适应症拓展空间** | ≥ 2 个适应症 | 峰值上限 |

## 壁垒判断

### 真壁垒
1. **专利保护（化合物专利 > 工艺专利 > 晶型专利）**：核心化合物专利是最强壁垒，保护 20 年。
2. **临床数据与监管先发**：首个获批的适应症 = 先发优势 + 医生心智。
3. **靶点/平台技术**：ADC 连接子技术、CAR-T 细胞制造工艺、mRNA 递送（LNP）——平台型技术可延伸多管线。
4. **商业化能力**（Pharma）：销售团队规模、医院覆盖、医保谈判能力——小 Biotech 的短板。
5. **生产壁垒（CGT 特别）**：CGT 的活细胞制造、病毒载体生产——CDMO 依赖度高，自建产能稀缺。

### 假壁垒（警惕）
1. **"me-too / me-better"管线扎堆**：PD-1 国内 100+ 家扎推——无差异化 = 价值归零。
2. **临床数据"统计学显著"但临床意义不大**：PFS 提升 0.5 个月 ≠ 商业成功。
3. **单一管线无平台延展性**：一旦失败无 Plan B。
4. **专利结构脆弱**：只有晶型/工艺专利，无化合物专利——仿制药可绕过。

## 红旗识别
1. **核心管线 Ⅱ 期数据不及预期**——估值腰斩起点。
2. **单一管线贡献 > 80% 估值**——风险未分散。
3. **现金 runway < 12 个月 + 无 BD 收入**——融资稀释风险。
4. **赛道严重扎堆（如 PD-1/CART）**——价格战 + 集采双重打击。
5. **核心专利即将到期（< 5 年）且无接续管线**——专利悬崖。
6. **CDE 否决/暂停临床**——监管黑天鹅。
7. **关键 opinion leader（KOL）公开质疑机制/数据**——学术风险。
8. **高管/首席医学官离职**——内部信息不对称信号。
9. **CDMO 单一供应商依赖（CGT）**——供应链风险。
10. **海外授权首付款远低于市场预期**——海外药企对该分子的真实判断（数据"注水"信号）。

## 占位符
- `[靶点/机制]`：PD-1/VEGF/CLDN18.2/BCMA 等。
- `[适应症]`：肿瘤/自免/代谢/罕见病 + 几线治疗。
- `[临床阶段]`：临床前/Ⅰ/Ⅱ/Ⅲ/NDA。
- `[成功率（PTS）]`：当前阶段的上市概率。
- `[销售峰值预测]`：患者数 × 渗透率 × 定价 × 用药时长。
- `[rNPV 计算表]`：各管线 rNPV 汇总。
- `[BD 交易清单]`：首付款/里程款/销售提成。
- `[专利到期日]`：核心化合物专利。
- `[现金 runway]`：月数。
