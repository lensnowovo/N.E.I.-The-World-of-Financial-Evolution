# 竞品分析：画清竞争格局和市场定位

这是 Anthropic 官方的 竞品分析 skill。系统梳理一个赛道或公司的竞争格局：

- 竞品清单：直接 / 间接 / 潜在竞争者

- 多维度对比：产品、定价、客户、融资、市场份额

- 定位图：把玩家映射到二维矩阵，看空白和拥挤区

- 壁垒与护城河分析

适合：研究新赛道、评估项目竞争位置的人。投前尽调和行业研究都用得上。
