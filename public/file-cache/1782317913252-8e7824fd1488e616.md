# LBO 杠杆收购模型：搭一套完整的收购估值

这是 Anthropic 官方的 LBO 杠杆收购模型 skill。搭一套完整的杠杆收购模型：

- 交易结构：资金来源（股权 + 各层债务）、收购价格

- 三大报表联动 + 债务偿还计划（SFP）

- 退出假设 + IRR / MOIC 收益测算

- 敏感性分析：进出倍数、债务成本对回报的影响

适合：做 PE 收购估值、考 LBO 建模 case 的人。配合 Excel 一起用。
