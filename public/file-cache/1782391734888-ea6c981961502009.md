# 新材料投资研究框架

## 这是什么

新材料产业（先进金属材料 / 高分子材料 / 无机非金属 / 复合材料 / 前沿纳米材料 / 生物基材料 / 电子化学品）的投资分析框架。新材料是"所有制造业的上游"——半导体、新能源、航空、生物医药的瓶颈往往卡在材料端。核心判断：**材料性能决定产业上限、自主可控决定生存底线、工程化能力决定商业化成败**。三大投资主线（替代 / 成长 / 前沿）优先级分明。适合 PE/VC 硬科技组、产业资本、政府引导基金、CVC（产业方战略投资）。

## 行业研究框架（6 维度）

### 维度 1：第一性原理（底层判断框架）

> 新材料投资的第一性原理——三个问题决定一个项目值不值得投。

| 原理 | 含义 | 投资判断 |
|---|---|---|
| **材料性能决定产业上限** | 下游产业的技术天花板由材料决定 | 性能未达标的材料 = 下游瓶颈 = 投资机会 |
| **自主可控决定生存底线** | 关键材料卡脖子 = 产业链断供风险 | 国产替代空间大 + 下游战略价值高 = 确定性机会 |
| **工程化能力决定商业化成败** | 实验室能做 ≠ 量产能做 ≠ 成本能做 | 工程化/量产能力是材料公司最被低估的壁垒 |

**关键认知**：
- 很多新材料"论文惊艳、量产艰难"——工程化能力（一致性、良率、成本）是真壁垒。
- 投材料不能只看"性能突破"，要看"性能 × 量产 × 成本"三者能否同时达标。

### 维度 2：四大底层逻辑（投资优先级）

| 底层逻辑 | 含义 | 优先级 | 代表材料 |
|---|---|---|---|
| **战略安全优先** | 关键材料断供影响国家安全 | ★★★★★ | 半导体材料、稀有金属、军工材料 |
| **国产替代确定性优先** | 已有明确需求 + 已验证技术路线 | ★★★★★ | 12 寸大硅片、光刻胶、电子特气 |
| **客户认证进度优先** | 已进入头部客户供应链 | ★★★★ | CMP 抛光液、靶材、碳纤维 |
| **绿色低碳优先** | 政策驱动 + 长期趋势 | ★★★ | 生物基材料、可降解材料、回收材料 |

**投资优先级**：战略安全 > 国产替代 > 客户认证 > 绿色低碳。

### 维度 3：三大投资主线（赛道选择）

> 新材料不是"一个赛道"，而是三条主线——判断项目属于哪条线，决定估值逻辑和退出节奏。

#### 主线 1：替代（国产替代）—— 确定性最高

**逻辑**：海外垄断 + 国产化率低 + 下游需求明确 → 国产替代的确定性机会。

| 材料类别 | 国产化率 | 海外垄断方 | 投资机会 |
|---|---|---|---|
| 12 寸大硅片 | ~20% | 信越/Sumco（日本） | ★★★★★ |
| 半导体光刻胶 | < 15% | JSR/东京应化/信越 | ★★★★★ |
| 电子特气 | ~30% | 空气化工/林德/大阳日酸 | ★★★★ |
| CMP 抛光液 | 20-30% | 卡博特/陶氏 | ★★★★ |
| 高端靶材 | ~40% | 日矿/霍尼韦尔 | ★★★ |
| 碳纤维（T800+） | < 30% | 东丽（日本） | ★★★★ |
| OLED 材料 | < 20% | UDC/默克 | ★★★★ |

#### 主线 2：成长（下游产业驱动）—— 弹性最大

**逻辑**：下游产业（新能源 / 半导体 / 航空 / 生物医药）爆发 → 上游材料需求量增长。

| 成长驱动 | 代表材料 | 成长逻辑 |
|---|---|---|
| **新能源车** | 正极材料（三元/磷酸铁锂）、电解液、隔膜、铜箔 | 电动车渗透率持续提升 |
| **新能源车 800V** | SiC 衬底/外延 | 高压平台刚需 |
| **光伏** | 硅料、POE 胶膜、银浆 | 装机量持续增长 |
| **AI/半导体** | HBM 材料、先进封装材料、电子化学品 | 算力需求驱动 |
| **商业航天** | 高性能合金、复合材料、耐高温材料 | 卫星互联网 + 可回收火箭 |
| **低空经济** | 碳纤维复合材料（eVTOL 机身） | eVTOL 量产 |
| **生物医药** | 医用高分子、生物相容材料 | 医疗器械国产化 |

#### 主线 3：前沿（颠覆性创新）—— 风险最高

**逻辑**：全新材料体系，实验室阶段，商业化时间表不确定。

| 前沿方向 | 成熟度 | 商业化时间 | 投资特征 |
|---|---|---|---|
| 石墨烯 | 中试阶段 | 5-10 年 | 应用场景分散，难聚焦 |
| 钙钛矿材料（光伏） | 中试阶段 | 3-5 年 | 效率提升快，稳定性待验证 |
| MOF（金属有机框架） | 实验室-中试 | 5-10 年 | 气体分离/储能潜力 |
| 超导材料（室温） | 纯实验室 | 10 年+ | 颠覆性但极不确定 |
| 纳米材料 | 部分应用 | 分领域 | 已有部分商业化 |

**主线优先级**：替代（确定性） > 成长（弹性） > 前沿（风险）。建议 60% 替代 + 30% 成长 + 10% 前沿。

### 维度 4：产品生命周期与投资时机

> 新材料从实验室到规模化经历清晰的 5 个阶段——不同阶段的投资逻辑、风险、估值完全不同。

| 阶段 | 标志 | 风险 | 估值方法 | 投资策略 |
|---|---|---|---|---|
| **1. 实验室研发** | 论文/专利 | ★★★★★ | 成本法 | 仅投顶尖科研团队 |
| **2. 中试** | 百吨级中试线 | ★★★★ | 风险调整 NPV | 关注工程化能力 |
| **3. 小批量量产** | 千吨级、首批客户认证 | ★★★ | PS / 可比交易 | 客户认证是关键 |
| **4. 规模化量产** | 万吨级、多客户 | ★★ | PE / EV/EBITDA | 看成本竞争力 |
| **5. 成熟 / 商品化** | 大宗化、价格竞争 | ★ | PE / 周期估值 | 看周期 + 成本优势 |

**投资时机判断**：
- **阶段 2→3 跨越（中试到小批量）**：风险最高但回报最高——**技术风险已大幅降低，但估值仍处于早期**。这是最佳投资窗口。
- **阶段 3→4 跨越（小批量到规模化）**：确定性高，但估值已反映——适合成长期 PE。
- **阶段 1（纯实验室）**：风险极高，除非团队极强，否则不适合财务投资。

### 维度 5：企业生命周期与估值

| 阶段 | 收入特征 | 估值方法 | 核心判断 |
|---|---|---|---|
| **研发期** | 无收入 | 成本法 + 团队溢价 | 团队 + IP + 技术路线 |
| **验证期** | 样品/小批量 | PS（极高倍数） | 客户认证进度 + 技术指标 |
| **成长期** | 规模化收入 | PS / PEG | 收入增速 + 毛利率 + 客户扩展 |
| **成熟期** | 利润稳定 | PE / EV/EBITDA | 成本优势 + 产能扩张 |

### 维度 6：中国特色——政策与产业基金

**政策驱动**：
- 新材料是"制造强国"战略的核心环节之一。
- 工信部"新材料首批次应用保险补偿"机制——降低下游试用国产新材料的风险。
- 各地新材料产业园 + 政府引导基金（宁波/淄博/连云港等化工新材料基地）。

**产业资本（CVC）主导**：
- 新材料投资中，**产业资本（下游大厂的战略投资）往往比纯财务 VC 更活跃**——因为材料验证需要下游配合，CVC 能提供场景 + 订单。
- 典型：宁德时代投正极材料、中芯国际投半导体材料、中航工业投航空材料。

## 行业核心指标

| 指标 | 健康阈值 | 说明 |
|---|---|---|
| **材料关键性能指标** | 达到或超越海外竞品 | 核心技术指标（因材料而异） |
| **量产规模** | 百吨级（特种）/ 万吨级（大宗） | 工程化能力体现 |
| **批次一致性（Cpk）** | > 1.33 | 量产稳定性的核心指标 |
| **单位成本** | 接近或低于海外竞品 | 商业化可行性 |
| **客户认证进度** | 已进入头部客户供应链 | **生死线** |
| **头部客户数量** | ≥ 3 家 | 避免单一客户依赖 |
| **毛利率** | > 25%（特种）/ > 15%（大宗） | 技术壁垒体现 |
| **研发费用率** | > 8% | 技术持续投入 |
| **产能利用率** | > 70% | 量产健康度 |
| **扩产计划** | 有明确产能扩张 | 成长信号 |
| **专利（发明专利占比）** | 发明专利 > 60% | IP 质量 |
| **CVC / 产业资本持股** | 有 | 场景 + 订单背书 |
| **政府首批次保险补偿** | 有 | 政策支持信号 |

## 壁垒判断

### 真壁垒
1. **工程化/量产 know-how**——实验室能做 ≠ 量产能做，一致性/良率/成本的工程经验是真壁垒。
2. **客户认证 + 长期供货关系**——材料认证周期长（1-3 年），一旦进入供应链粘性极强。
3. **核心配方/工艺专利**——材料的核心壁垒常在"配方"和"工艺参数"，难以逆向。
4. **规模效应（特种材料也适用）**——量产规模带来成本优势 + 质量稳定性。
5. **稀缺原材料资源**——稀土、特定矿产的掌控（中国稀土/钨/锑等资源优势）。
6. **CVC 下游场景绑定**——下游大厂的战略投资 = 稳定订单 + 场景验证。

### 假壁垒（警惕）
1. **"论文 Nature/Science"但无量产能力**——学术突破 ≠ 商业化能力。
2. **"性能超越海外"但成本高 3-5 倍**——性能达标但成本不达标 = 无法替代。
3. **"国产替代空间大"但无客户认证**——概念性指标，非真实市占。
4. **"专利数量多"但发明专利占比低**——实用新型/外观专利 ≠ 技术壁垒。
5. **"首批次应用保险"补贴占收入 > 30%**——政策依赖，非真实商业能力。
6. **"绿色低碳"概念但无下游需求**——政策概念 ≠ 商业需求。

## 红旗识别
1. **实验室性能惊艳但无中试数据**——工程化能力未验证。
2. **宣称"对标海外龙头"但成本高 3 倍以上**——成本不达标无法替代。
3. **单一客户占比 > 70% 且无扩展计划**——客户集中风险。
4. **量产规模长期停在"百公斤级"**——工程化能力不足。
5. **批次一致性差（Cpk < 1.0）**——量产质量不稳定。
6. **管理层纯学术背景、无产业经验**——材料量产需要工程化经验。
7. **"绿色低碳"叙事但无付费客户**——政策概念而非商业需求。
8. **首批次保险补贴占收入 > 30%**——政策依赖。
9. **专利以实用新型为主（发明专利 < 40%）**——IP 质量低。
10. **无 CVC / 产业资本背书**——材料投资无下游场景配合风险极高。
11. **估值 PS > 15x 但收入 < 1 亿**——纯叙事估值。
12. **宣称"颠覆性新材料"但无中试线**——纯实验室阶段，风险极高。
13. **原材料单一来源且无备选方案**——供应链风险。
14. **环保/安全合规记录差**——化工类材料环保风险高。
