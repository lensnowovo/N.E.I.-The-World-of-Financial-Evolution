# 基金运营管理 SOP · Skill 草稿

## 这是什么

面向 PE/VC GP（基金管理人）团队、IR、运营人员的**基金运营管理全流程 SOP**。基金运营是"把基金当成一家公司来管"——从募集设立到最终清算，覆盖募（LP 开发/基金设计/路演）、投（投决流程/风险控制）、管（投后/LP 报告/估值）、退（退出路径/S 交易/收益分配）、清（基金清算/合规归档）五大环节。这不是投资研究方法论，而是**基金作为法人实体的运营工作流**——GP 合伙人、运营总监、IR/中后台必读。一个管理规模 50 亿的基金，运营失误可能比单个项目亏损更致命。适合 GP 合伙人、运营总监、IR、基金财务、法务合规、外包的基金服务商（律所/审计/托管）。

## 方法论框架（6 维度）

### 维度 1：基金生命周期全流程（5 阶段）

> 基金是一个有生命周期的法人实体——5 阶段各有标准化工作流。

| 阶段 | 时长 | 核心动作 | 主导部门 |
|---|---|---|---|
| **① 设立募集** | 6-18 月 | 基金设计 / LP 开发 / 路演 / 首关 / 终关 | IR + 合伙人 |
| **② 投资期** | 3-5 年 | 项目开发 / 尽调 / 投决 / 交割 | 投资团队 |
| **③ 投后期** | 与投资期并行 | 投后管理 / LP 报告 / 估值调整 | 投后 + IR |
| **④ 退出期** | 3-7 年 | IPO / 并购 / S 交易 / 回购 | 投资团队 + 财务 |
| **⑤ 清算期** | 1-2 年 | 基金清算 / 收益分配 / 合规归档 | 财务 + 法务 |

**典型存续期**：
- VC 基金：7-10 年（投资期 3-5 + 退出期 4-5）。
- PE 基金：7-8 年。
- 政府引导基金：10-15 年（鼓励耐心资本）。
- 可延期：1-2 年（需 LP 大会通过）。

### 维度 2：LP/GP/管理人/托管四方关系

> 基金运营的底层是"四方关系"——理解各自权利义务才能避免运营事故。

```
┌──────────────────────────────────────────┐
│  LP（有限合伙人）= 出钱方                 │
│  权利：知情权/收益权/咨询委员会/退出权    │
│  义务：按约定缴款 / 不参与日常管理        │
└──────────────────┬───────────────────────┘
                   │ 信任委托
┌──────────────────▼───────────────────────┐
│  GP（普通合伙人）= 管钱方                 │
│  职责：投资决策 / 基金管理 / 信义义务     │
│  收入：管理费 + 超额收益分成（Carry）     │
└──────────────────┬───────────────────────┘
                   │ 实际执行
┌──────────────────▼───────────────────────┐
│  基金管理人（管理牌照）                   │
│  在中国：中基协登记的私募基金管理人       │
│  合规要求：专业化经营/防火墙/信义义务     │
└──────────────────┬───────────────────────┘
                   │ 资金托管
┌──────────────────▼───────────────────────┐
│  托管机构（银行/券商）                    │
│  职责：资金托管/资金划拨监督/账务核算     │
└──────────────────────────────────────────┘
```

#### 关键文件

- **LPA（有限合伙协议）**：基金的根本大法——约定投资范围、GP 权限、收益分配、LP 权利。
- **Side Letter（附属协议）**：大 LP 单独约定的特殊条款（最惠国待遇/特别知情权/费用减免）。
- **Subscription Agreement（认购协议）**：LP 承诺出资的法律文件。
- **PPM（Private Placement Memorandum）**：基金募集说明书——披露策略/团队/风险/费用。

### 维度 3：LP 管理 IR 工作流（最容易被低估的环节）

> IR（Investor Relations）不是"融资业务员"，是基金运营的核心枢纽——管好 LP 关系是基金能否持续募资的关键。

#### IR 日常工作 4 大模块

##### 1. LP 沟通节奏

| 沟通 | 频率 | 内容 |
|---|---|---|
| 季度报告 | 季末 + 45 天 | 基金业绩 / 投资进展 / 退出动态 / 市场展望 |
| 年度报告 | 年度 + 90 天 | 完整年度业绩 / 估值 / 审计 / 税务 |
| LP 大会（AGM） | 年度 1 次 | 全体 LP / 战略汇报 / 投票事项 |
| LP 咨询委员会（LPAC） | 季度 / 半年 | 利益冲突事项 / 重大决策咨询 |
| 临时重大事项 | 事件驱动 | 重大投资 / 退出 / 合规事件 |

##### 2. LP 现金流管理

- **Call 款（Capital Call）**：GP 向 LP 发起缴款通知——用于投资 / 管理费 / 费用。
- **Distribution（分配）**：项目退出后向 LP 分配现金或股票。
- **LP 资金利用率**：实缴 / 认缴——LP 最关注的指标之一，反映基金效率。
- **Call 款预警**：提前 10-15 个工作日通知 LP——给 LP 时间调资金，避免违约。

##### 3. LP 期望管理

- LP 类型差异极大：**国资 LP（合规重/流程慢）** / **险资（长线/稳重）** / **母基金（专业/挑剔）** / **家族办公室（灵活）** / **高净值（敏感/沟通密集）**。
- 不同类型 LP 的"痛点"完全不同——IR 要分别制定沟通策略。
- **"坏消息早说"原则**——项目出问题主动告知，远好过 LP 在审计中发现。

##### 4. LP 续募闭环

- LP 复投率是基金可持续性的核心指标——下一期基金 60% 以上 LP 来自现有 LP 是健康信号。
- IR 要在退出期就开始为下一期基金铺垫——而非等清算才开始募资。

### 维度 4：投资决策流程（投决 SOP）

> 标准化的投决流程是基金治理的基础——避免"一言堂"和个人决策风险。

#### 典型投决 5 步流程

```
项目立项 → 投资团队初筛 → IC 投决会审议 → 合伙人决策 → 投决执行
   │            │              │              │             │
   └ 投资备忘录  └ 尽调报告     └ IC 票决     └ 终审        └ 签约交割
```

#### 投决委员会（IC）规则

- **成员**：3-7 人（合伙人 + 外部专家，奇数便于票决）。
- **回避制度**：有利益冲突的成员回避。
- **票决规则**：多数决（> 50%）或绝对多数（≥ 2/3），按 LPA 约定。
- **IC 备忘录**：标准模板（项目摘要 / 投资逻辑 / 尽调结论 / 风险点 / 估值 / 退出路径）。
- **投后追溯**：每个项目投后回顾，IC 决策 vs 实际结果的偏差分析——持续优化投决。

#### 风险控制"三道防线"

| 防线 | 角色 | 职责 |
|---|---|---|
| **第一道** | 投资团队 | 项目尽调 / 风险识别 |
| **第二道** | 风控/合规 | 合规审查 / 风险评估 |
| **第三道** | 外部审计/法律 | 独立审计 / 法律意见 |

### 维度 5：基金财务与税务

#### 基金费用结构

| 费用类型 | 标准 | 说明 |
|---|---|---|
| **管理费** | 认缴 2-2.5% / 年（VC） / 实缴 1.5-2% / 年（PE） | 投资期内收，退出期递减 |
| **超额收益分成（Carry）** | 20%（行业标准） | 超过门槛收益部分 |
| **门槛收益（Hurdle）** | 6-8% / 年 | LP 优先收益 |
| **Catch-up** | 100% / 50% / 0 | GP 追赶机制 |
| **追回条款（Clawback）** | 有 | 历史多分配由 GP 退还 |

#### 基金会计核心

- **估值**：每个季度更新估值——按 IPEV（国际私募股权估值理事会）准则。
- **Funded 比例**：实缴 / 认缴。
- **DPI（Distributed to Paid-In）**：已分配 / 已实缴——LP 最看重的指标。
- **TVPI**（Total Value to Paid-In）：总价值 / 已实缴。
- **IRR**：内部收益率——但易被操纵，需结合 DPI/TVPI 综合看。
- **MOIC**（Multiple of Invested Capital）：投资倍数。

#### 基金税务

- 中国有限合伙基金：**先分后税**——LP 按各自类型交税。
- 公司型基金：双重征税（基金公司层面 + LP 层面）。
- 创投税收优惠：财税 55 号文——符合条件的创投可按 20% 缴个税。

### 维度 6：S 交易与基金流动性管理

> S（Secondary）交易是 LP 和 GP 的新流动性工具——2020 年后中国市场快速增长。

#### S 交易 4 种形态

| 类型 | 含义 | 发起方 |
|---|---|---|
| **LP 份额转让** | LP 将份额卖给 S 基金 | LP |
| **GP-led 接续基金** | GP 设立新基金收购老基金资产 | GP |
| **直接二级（Direct Secondary）** | 收购 LP 持有的底层资产组合 | S 基金 |
| **优先股结构** | S 基金以优先股形式注入流动性 | GP + LP |

#### S 交易触发场景

- LP 流动性需求（自身的偿债/出资压力）。
- 基金到期延期/清算，但部分资产未退出。
- GP 想延续明星资产的持有（接续基金）。
- LP 结构调整（国资 LP 退出）。

## 行业核心指标

| 指标 | 健康阈值 | 说明 |
|---|---|---|
| **LP 续投率** | > 60% | 下一期基金可持续性 |
| **LP 资金利用率** | > 80%（投资期内） | 基金效率 |
| **DPI@5 年** | > 1.0x | LP 现金回报 |
| **TVPI** | > 2.0x（成熟基金） | 总价值倍数 |
| **IRR@清算** | > 15%（PE）/ > 20%（VC） | 内部收益率 |
| **管理费收缴率** | 100% | LP 违约率控制 |
| **季度报告准时率** | 100%（季末 + 45 天） | IR 质量 |
| **投决命中率** | > 60% | 投决有效性 |
| **LP 违约率** | < 2% | LP 资质筛选 |
| **LPAC 参与率** | > 80% | LP 治理参与度 |

## 红旗识别
1. **IR = 募资业务员**——LP 关系只募资不维护，续募断层。
2. **季度报告延迟 > 90 天**——LP 信息滞后，信任受损。
3. **估值长期不更新**——"按成本法估值"掩盖问题。
4. **LP Call 款前 3 天才通知**——LP 调头寸紧张，违约风险。
5. **无投决回避制度**——利益冲突项目无防火墙。
6. **管理费长期不递减**——投资期后仍按 2% 收，LP 不满。
7. **DPI 5 年仍 < 0.5x**——LP 现金回报差，续募困难。
8. **Carry 分配在 DPI < 1 时启动**——LP 未回本就分 GP，违反行业惯例。
9. **LP 违约率 > 5%**——LP 资质筛选不严。
10. **LPAC 形同虚设**——利益冲突事项无审查机制。
11. **无追回条款（Clawback）**——历史多分配无法追回，LP 损失。
12. **LPA 与实际操作不符**——合同条款 vs 执行脱节。
13. **无 S 交易预案**——基金到期部分资产无退出，被动延期。
14. **国资 LP 未做特別合规**——国资 LP 退出/审计要求特殊。
