# 市场规模测算 SOP：TAM / SAM / SOM + 自上而下与自下而上

## 这是什么
一套面向 **产品经理、创业者、早期投资人** 的市场规模测算方法。解决"这个产品/赛道的市场到底有多大"这个 BP 和立项报告里必答却最容易拍脑袋的问题。用结构化的 TAM/SAM/SOM 三层 + 自上而下/自下而上两法，把"感觉很大"变成"可验证的数字"。

## 方法论框架

### 第一步：定义你的目标受众与可服务市场总量（TAM）
**TAM（Total Addressable Market）= 理论上你能触达的所有用户规模（理想极限）**

操作：
1. 列出所有可能消费该产品/服务的人群类型。
2. 用人口统计特征（年龄、性别、地理位置、社会经济背景）初步框定。
3. **关键动作**：与现有客户/潜在客户直接沟通，了解"为什么购买"——而非假设"所有人都会买"。

> 反例：发明一款卡路里追踪可穿戴设备，假设 TAM = "所有美国成年人" → 天真。真实 TAM = 想要更健康生活方式的人群 + 需要控制饮食的运动人群 + 雇主/健康保险公司。

### 第二步：收集市场数据，确定 SAM 和 SOM

**SAM（Serviceable Addressable Market）= 现阶段可实际覆盖到的市场**
- 受限于地理、渠道、语言、法规等。
- 例：能量饮料只在英格兰南部某超市销售 → SAM = TAM 中位于英格兰南部的人。

**SOM（Serviceable Obtainable Market）= 预计能实际拿下的市场份额**
- 通过营销和销售努力能争取到的最小子集。
- 典型 SOM 客户：SAM 中未被良好服务的 + 对现有产品不满意的 + 愿尝试新版本的人群。

### 第三步：收集更广泛的市场规模信息（5 类来源）

#### 3.1 行业协会数据
- 几乎每个行业都有正式协会，会发布行业规模、年支出、平均零售价。
- **中国特色补充**：中国 xx 行业协会、中国 xx 联合会——但要注意数据口径和水分。

#### 3.2 竞品市场地图
- 将竞品按价格高低/质量高低排列在坐标轴上。
- **理想结果**：凸显你的产品是否处于独特细分市场（如"质量最好但定价较低"= 差异化优势）。

#### 3.3 消费者研究
- 用焦点小组、深度访谈验证需求真实性。
- **关键检验**：如果发现市场空白，必须搞清楚"为什么没人填补"——是机会还是需求根本不存在。

#### 3.4 品牌感知研究
- 消费者对你品牌的真实认知（不是你希望的认知）。
- 影响能吸引的客户类型、定价水平、合作可能性。

#### 3.5 市场调研问卷
- 在线问卷工具可覆盖全球、快速生成数据。
- 缩小目标市场规模后用于验证假设。

### 第四步：使用市场规模计算公式

**核心公式**：
```
潜在客户总数 × 平均单个客户收入 = 市场规模
```

**进阶公式**（考虑购买频率）：
```
潜在客户数 × 单次购买量 × 购买频率 × 单价 = 市场规模
```

**份额估算法**（替代）：
```
行业年价值 × 预期市场份额（通常 1%–5%）= 你的市场规模
```

**示例**：祛痘洗面奶
- 50 万潜在客户 × 12 月 × 10 英镑/支 = **6000 万英镑/年**。
- 或：洗面奶行业年价值 10 亿英镑 × 1%–5% 份额 = **1000 万–5000 万英镑/年**。

### 第五步：展望未来（1/2/3 年预测）
- 历史数据分析：市场是上升轨迹还是萎缩？
- 客户群是否会扩大？（用人口/收入/政策趋势推演）
- 提供 1 年、2 年、3 年的规模预测。
- 考虑分销拓展和新地区扩张。
- **关键问题**：产品的颠覆性潜力是什么？（数码相机 vs 胶片、Kindle vs 实体书店）
- **跟踪方法**：查看正在申请的专利、阅读行业媒体、设置 Google 提醒追踪关键词。

## 自上而下 vs 自下而上：何时用哪个

| 方法 | 逻辑 | 优点 | 缺点 | 适用场景 |
|---|---|---|---|---|
| **自上而下** | 从行业整体 → 推到自身份额 | 快速、数据易得 | 过于宽泛，缺乏洞察 | 早期判断、BP 初稿 |
| **自下而上** | 从自身细分数据 → 推到行业趋势 | 真实可靠，可指导战略 | 耗时、需研究能力 | 立项前深度验证、战略决策 |

**建议**：早期用自上而下快速判断，立项前用自下而上深度验证。

## 投资人预期
- **大多数 VC/天使投资人希望潜在市场规模 ≥ 10 亿英镑（或等值）**。
- **如果数据低于此**：不要虚报。解释为什么相信市场潜力，或为什么这个产品值得做（如让世界更好）。
- **投资人理念各不相同**：做好研究，总能找到适合的投资者。

## 占位符
- `[产品/服务名称]`
- `[目标受众画像]`：年龄、性别、地域、收入、行为特征。
- `[TAM 用户数]`：理论上所有可能用户数。
- `[SAM 用户数]`：现阶段可触达用户数（地理/渠道/法规限制后）。
- `[SOM 用户数]`：预期能拿下的份额。
- `[单客户 ARPU]`：单客户平均收入。
- `[购买频率]`：每月/每年购买次数。
- `[1/2/3 年规模预测]`：含增速假设。
- `[颠覆性潜力描述]`：产品对行业的潜在颠覆。

## 红旗识别（这些信号说明测算可能有问题）
1. **TAM 定义过宽**："所有 xx 人群"——天真假设，缺乏细分。
2. **只做自上而下，未做自下而上验证**——数字虚高。
3. **SAM 与 SOM 混淆**：把可触达市场当成能拿下的市场。
4. **购买频率假设脱离实际**：消耗品按一次性算，或耐用品按高频算。
5. **未考虑间接竞争**：只看同类产品，忽视替代方案（如外卖 vs 方便面）。
6. **历史数据老化**：用 3 年前的行业数据预测今天的市场。
7. **未解释"市场空白为什么存在"**：可能空白的背后是需求不存在，而非机会。
8. **增长率假设过于乐观**：3 年 CAGR 假设 > 50% 而无强驱动逻辑。
9. **未做敏感性分析**：只给一个数字，没有区间（乐观/中性/悲观）。
10. **数据来源单一且无可追溯性**：无法验证。
