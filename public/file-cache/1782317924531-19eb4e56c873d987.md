# 行业概览：快速摸清一个赛道的全景

这是 Anthropic 官方的 行业概览 skill。快速产出一份赛道的全景研究：

- 市场规模与增速（TAM / SAM）

- 价值链与产业链上下游

- 主要玩家与竞争格局

- 关键驱动因素与风险

- 投资机会与值得关注的方向

适合：刚接触一个新赛道、要做行业研究的人。投前研究、LP 汇报都用得上。
