# Process Letter 流程信：给买家的竞标规则说明

这是 Anthropic 官方的 Process Letter（流程信） skill。卖方拍卖流程中发给买家的规则说明：

- 时间表：各轮报价截止日、管理层会面安排

- 报价要求：估值方法、对价结构、条件

- 尽调安排：数据室访问、Q&A 流程

- 规则约束：保密、禁止挖人、交易保护条款

适合：FA 组织竞标拍卖流程。流程信是拍卖有序进行的关键文档。
