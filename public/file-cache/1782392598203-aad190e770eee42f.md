# 政策检索方法论 · Skill 草稿

## 这是什么

面向 PE/VC/FA 投研人员的**中国政策信息检索与情报分析 SOP**。中国投资中，政策往往是最大的变量——产业政策决定行业天花板、监管政策决定商业模式合法性、补贴政策影响短期财务、地方政策影响落地节奏。但很多投研人查政策只会"百度一下"，找不到原文、漏掉征求意见稿、看不懂文号、不会搭建监测体系。本 Skill 把政策检索拆成"找原文 → 看懂 → 追踪变化 → 搭监测体系"四步，可作为任何行业研究的前置动作。适合中国一级/二级市场投研、产业资本、政策研究、战略咨询。

## 方法论框架（6 维度）

### 维度 1：政策信息源全景图（去哪找）

**4 层信息源体系**——按权威性 + 时效性分层：

| 层级 | 平台 | 特点 | 用途 |
|---|---|---|---|
| **L1 国家级权威平台** | 中国政府网 gov.cn / 国家法律法规数据库 flk.npc.gov.cn / 全国人大网 npc.gov.cn / 国务院客户端 | 最高权威、原文 | 法律法规原文、国务院政策 |
| **L2 部委专业平台** | 发改委 ndrc / 工信部 miit / 财政部 mof / 央行 pbc / 国家统计局 / 科技部 / 商务部 / 卫健委 / 生态环境部 / 国家能源局 / 农业农村部 | 部委规章、征求意见稿、实施细则 | 产业政策、行业准入、补贴细则 |
| **L3 专项政策平台** | 国家中小企业政策信息平台 sme-service.cn / 自贸区服务网 ftz.mofcom / 一带一路官网 / 国家知识产权局 cnipa | 专题集纳 | 专项研究 |
| **L4 地方政府平台** | 省级政府门户（格式：www.[省份拼音].gov.cn，如 gd.gov.cn / zj.gov.cn / shanghai.gov.cn）→ 政务公开/政策文件 | 地方落地、补贴申领 | 落地节奏、地方差异 |

**关键认知**：
- 同一政策往往在**多个层级发布**——国务院定方向、部委出细则、地方出执行方案。三层都要查全。
- **征求意见稿 > 正式稿**：征求意见稿是政策信号的前哨，比正式发布早 3-12 个月。

### 维度 2：检索技巧（怎么找）

#### 高级搜索运算符速查表

| 运算符 | 用法 | 示例 |
|---|---|---|
| `""` | 精确匹配短语 | `"新质生产力"` |
| `-` | 排除关键词 | `苹果 -水果` |
| `site:` | 限定网站 | `碳达峰 site:gov.cn` |
| `filetype:` | 限定文件格式 | `十四五规划 filetype:pdf` |
| `intitle:` | 标题含关键词 | `intitle:实施意见` |
| `inurl:` | URL 含关键词 | `inurl:policy` |
| `OR` / `\|` | 多关键词 | `减税 OR 降费` |
| `*` | 通配符 | `"*五规划"` |

#### 投研场景检索模板（4 个高频场景）

```bash
# 场景 1：找部委政策原文（PDF）
新能源 site:ndrc.gov.cn filetype:pdf

# 场景 2：精确找"实施意见"类正式文件
"专精特新" intitle:实施意见 site:gov.cn

# 场景 3：跨部委查政策（近一年）
碳中和 2024 site:mee.gov.cn OR site:ndrc.gov.cn

# 场景 4：排除新闻报道，仅要政策原文
数字经济 site:gov.cn -新闻 -报道 filetype:pdf
```

#### 政府网站高级筛选（绝大多数 gov.cn 都支持）

筛选维度：关键词（全文/标题） / 发布时间 / 文号 / 发文机关 / 主题分类 / 公文类型（通知/意见/办法/规划） / 有效性（现行有效/已废止/已修订）。

### 维度 3：政策文号解读（看懂层级）

```
国发〔2024〕15号
  │    │    └── 序号：当年该机关第 15 份发文
  │    └── 年份：2024 年
  └── 发文机关简称：国务院
```

**常见发文机关简称与层级**：

| 简称 | 机关 | 政策层级 |
|---|---|---|
| 中发 | 中共中央 | 最高 |
| 国发 | 国务院 | 国家级 |
| 国办发 | 国务院办公厅 | 国家级 |
| 发改〔领域〕 | 国家发改委 | 部委级 |
| 财税 | 财政部、税务总局 | 部委级 |
| X 政发 | 省/市政府 | 地方级 |

**层级含义**：中发 > 国发 > 部委 > 地方。投资判断时要看清是哪个层级发的——同样是"鼓励"，国务院文件和某司局发言分量天差地别。

### 维度 4：政策解读框架（看懂意图）

> 拿到政策原文后，按 8 维度框架拆解，避免"只看到标题、错过细节"。

1. **政策背景**：为什么出台？解决什么问题？
2. **核心要点**：说了什么？有哪些关键措施？
3. **适用范围**：谁适用？什么领域？什么时间？
4. **量化指标**：有哪些具体数字和目标？（这部分最关键，直接影响行业测算）
5. **配套措施**：有哪些保障和支持？（财政/税收/金融/土地）
6. **责任分工**：谁负责？谁监督？（决定执行力度）
7. **时间节点**：何时生效？有无过渡期？
8. **影响评估**：对行业/企业/个人有何影响？（合规成本/政策红利/准入门槛）

### 维度 5：政策信号重要性排序与趋势研判

> 不是所有"政策信号"分量都一样——按以下顺序判断真实力度。

| 信号源 | 重要性 | 含义 |
|---|---|---|
| 🔴 中央重要会议决定 / 中央文件 | 最高 | 战略方向，必执行 |
| 🟠 国务院常务会议 / 国务院文件 | 很高 | 落实路径明确 |
| 🟡 部委征求意见稿 / 试点方案 | 中高 | 前哨信号，3-12 个月后正式出台 |
| 🟢 领导讲话 / 媒体报道 | 参考 | 风向，但未必落地 |

**趋势研判要点**：
- 征求意见稿发布 → 试点城市/企业 → 正式文件 → 配套细则 → 地方落地，这是一条完整的政策传导链，投研要顺着这条链追踪。
- **两会（每年 3 月）、中央经济工作会议（每年 12 月）、中央全会（通常 10 月）**是三大政策密集节点，要提前关注。

### 维度 6：政策监测与情报网络搭建（持续追踪）

#### 监测工具矩阵

| 工具类型 | 代表工具 | 适用场景 |
|---|---|---|
| 官方订阅 | 政府网站 RSS / 邮件订阅 | 特定部门追踪 |
| 搜索引擎快讯 | Google Alerts / 百度订阅 | 全网关键词监测 |
| 专业政策平台 | 人民数据政策库 / 威科先行 | 系统性研究 |
| 舆情监测系统 | 各类舆情平台 | 政策舆论分析 |
| AI 辅助 | ChatGPT / Claude 文本分析 | 政策文本深度解读、新旧对比 |

#### 政策预警机制（4 要素）

| 要素 | 说明 | 示例 |
|---|---|---|
| **警素（指标）** | 监测的具体指标 | 环保标准变化、税率调整 |
| **警兆（先兆）** | 政策变化的前兆信号 | 征求意见稿发布、领导讲话 |
| **警度（级别）** | 预警分级 | 红/橙/黄/蓝 |
| **警限（阈值）** | 触发条件 | 政策草案发布后 30 天 |

```markdown

| 序号 | 来源 | 网址 | 频率 |
|---|---|---|---|
| 1 | 国家发改委 | ndrc.gov.cn | 每日 |
| 2 | 工信部 | miit.gov.cn | 每日 |

### 二、关键词列表

```

## 行业核心指标

| 指标 | 健康阈值 | 说明 |
|---|---|---|
| **政策原文获取完整度** | 国务院+部委+地方三层齐全 | 不能只看媒体报道 |
| **征求意见稿追踪率** | 覆盖核心部委 | 抢占 3-12 个月提前量 |
| **政策解读 8 维度完整度** | 8 维度全部覆盖 | 避免漏看量化指标 |
| **监测体系覆盖** | 核心 3-5 源 + 关键词订阅 | 持续追踪 |
| **政策传导链追踪** | 征求意见→试点→正式→细则→地方 | 完整链条 |
| **预警响应时长** | 征求意见稿发布后 ≤ 7 天 | 抢占研判窗口 |

## 红旗识别
1. **只看媒体报道、不查原文**——媒体报道常断章取义，必须查 gov.cn 原文。
2. **只看国务院文件、漏掉部委细则**——部委细则才是落地执行依据。
3. **不知道文号含义**——国发/中发/发改简称分不清层级。
4. **把领导讲话等同于政策**——讲话是风向，未必落地。
5. **不追踪征求意见稿**——等正式发布再研究就晚了。
6. **不区分"现行有效/已废止/已修订"**——用了已废止政策做判断。
7. **只看国家级、不看地方落地**——地方执行差异巨大。
8. **不会用 site:/filetype: 高级搜索**——检索效率极低。
9. **没有监测体系，靠临时抱佛脚**——被动反应而非主动研判。
10. **不追踪政策传导链**——征求意见→试点→正式→细则→地方，链条断裂。

## 占位符
- `[行业名称]`：新能源 / 半导体 / 医疗 / 教育 / 金融……
- `[核心业务]`：补贴 / 准入 / 标准 / 监管 / 税收优惠
- `[发文机关]`：国务院 / 发改委 / 工信部 / 央行 / 地方政府
- `[文号]`：国发〔YYYY〕XX 号 / X 政发
- `[政策传导阶段]`：征求意见 / 试点 / 正式发布 / 配套细则 / 地方落地
- `[量化指标]`：政策中的具体数字（装机目标、补贴金额、税率）
- `[时间节点]`：生效日 / 过渡期 / 试点截止
