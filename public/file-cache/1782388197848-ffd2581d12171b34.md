# 出海 / 全球化投资研究框架

## 这是什么
中国企业出海（品牌出海/产品出海/服务出海/技术出海）的投资分析框架。适合 **PE/VC/产业资本** 在"不出海就出局"共识下做出海赛道判断与尽调。2026 年的核心范式转变：**技术平权抹平了传统壁垒，估值从"技术想象力"转向"商业落地真实度"**——投资人从单一技术壁垒转向 4 大核心指标：场景落地真实性、存量资源复用能力、现金流稳定性、全球化组织承载能力。

## 行业研究框架（6 维度）

### 维度 1：4 大出海模式判断

| 模式 | 特征 | 代表 | 投资关注点 |
|---|---|---|---|
| **产品出海**（贸易型） | 制造能力输出，OEM/ODM 为主 | 光伏组件、动力电池、家电 | 产能、成本、贸易壁垒 |
| **品牌出海** | 品牌力 + 本地化营销 | 大疆/安克/SHEIN/名创优品 | 品牌势能、本地化、渠道 |
| **技术出海** | 技术平台/API/SaaS 输出 | AI 应用、开发者工具 | 场景真实性、付费意愿、留存 |
| **服务/生态出海** | 产业链整体迁移 + 本地化运营 | 跨境支付、物流、电商运营 | 合规、本地资源、网络效应 |

**投资判断**：从"产品出海"向"品牌出海"转型是高溢价方向（恒旭资本观点）。

### 维度 2：4 大核心估值维度（2026 新标准）

> 这是 2026 年 VC 评判出海项目的全新底层逻辑，取代了过去的技术估值法。

#### 指标 1：场景落地真实性
- **B 端**：是否瞄准海外社媒内容/跨境营销/私域运营/智能商务对接等真实刚需。
- **C 端**：是否深耕垂直细分赛道 + 配套培训/社群服务——跳出通用工具同质化。
- **红旗**：无真实场景、无行业资源、无稳定营收的纯概念项目。

#### 指标 2：存量资源复用能力（B 端核心）
- 团队长期深耕积累的人脉圈层、合作生态、招商渠道、会员体系、销售网络。
- 这些是业务落地的根基，直接决定推进效率与营收上限。
- B 端项目依托垂直行业存量资源可持续变现，抗周期抗风险能力强。

#### 指标 3：现金流稳定性
- 优质出海生意的 3 个特征：需求被真实流量验证 + 变现模式有差异化优势 + 后端服务有可持续技术支撑。
- 告别"烧钱换规模"，转向运营效率、商业模式健康度、盈利确定性。
- **B 端商业模式落地路径**：垂直行业 AI 增值赋能（轻资产高毛利）+ 企业私有化智能体（长期产业价值）。

#### 指标 4：全球化组织承载能力
- 跨文化管理、跨境团队协同、本地化人才招聘留存——最难构建的壁垒。
- 核心团队协同失衡直接影响落地效果和服务口碑。
- 检查项：是否有专职海外属地团队？还是仅国内统筹 + 海外 KOL 邮件沟通？

### 维度 3：B 端 vs C 端出海逻辑对比

| 维度 | B 端出海（2026 资本共识优选） | C 端出海 |
|---|---|---|
| **需求特性** | 刚性、复购稳定、模式可沉淀 | 依赖公域流量，波动大 |
| **获客** | 存量资源变现 | 依赖买量，CAC 持续走高 |
| **壁垒** | 资源积淀 + 行业 know-how | 独家内容/版权资产 + 信任体系 |
| **交付模式** | "AI + 标准化服务 + 落地交付" | 工具 + 内容 + 增值服务 |
| **估值** | 现金流稳定，估值溢价高 | 同质化内卷，估值压缩 |

**B 端交付模式的核心矛盾**：纯非标定制无法规模化；纯标准化产品无差异化。被资本认可的模式：**标准化产品套餐 + 体系化交付能力**。

### 维度 4：本地化深度（品牌出海成败关键）

> **本地化是品牌出海最大的软门槛**。产品可以标准化，但品牌必须与当地文化、价值观、生活方式共鸣。

**本地化深度 4 层次**：
1. **L1 产品本地化**：功能适配（OPPO 优化拉美拍摄/印尼信号弱攻关）。
2. **L2 营销本地化**：本地 KOL、本地化内容、本地社交平台。
3. **L3 组织本地化**：本地团队、本地管理层、本地决策权。
4. **L4 文化共鸣**：融入当地生活方式，从"卖产品"到"融入生态"。

**投资判断**：L1/L2 是基础，L3/L4 是真壁垒。仅停留在 L1/L2 的项目难穿越周期。

### 维度 5：合规与政策风险

**重点核查**：
- **数据合规**：跨境数据传输（GDPR/CCPA/中国数据安全法）。
- **本地化法规**：印尼 POS法案、印度数据本地化、欧盟 DSA。
- **贸易壁垒**：美国对华关税、印度封禁、欧盟碳关税。
- **本地公司架构**：VIE/合资/独资——决定控制权和退出路径。
- **税务架构**：BEPS 2.0 全球最低税、转让定价。

### 维度 6：估值重估机会

**4 个重估维度**（投资机构视角）：

| 维度 | 对应传统估值因子 | 出海溢价来源 |
|---|---|---|
| **出海规模** | 市场空间（TAM） | 从单一国家 → 全球市场 |
| **全球价值链地位** | 盈利质量 | 从代工低毛利 → 品牌高毛利 |
| **出海竞争壁垒** | 行业竞争格局 | 本地化 + 组织能力的稀缺 |
| **抗风险能力** | 长期发展价值 | 多市场分散单一国家风险 |

## 行业核心指标

| 指标 | B 端 | C 端 | 说明 |
|---|---|---|---|
| **海外营收占比** | > 50% | > 70% | 是否真出海 |
| **海外营收增速** | > 50% YoY | > 100% YoY | 成长性 |
| **NRR（净收入留存率）** | > 110% | — | B 端核心 |
| **CAC / LTV** | LTV/CAC > 3 | LTV/CAC > 2 | 单位经济 |
| **海外毛利率** | > 60%（AI/SaaS） | > 40%（品牌） | 高于国内同类 |
| **海外团队占比** | > 30% | > 20% | 本地化深度 |
| **复购/续费率** | > 80% | > 40% | |
| **海外 NPS** | > 40 | > 30 | 用户口碑 |
| **国库/现金流 runway** | > 18 个月 | > 12 个月 | |
| **合规事件数** | 0 | 0 | 任何监管处罚都是风险 |

## 壁垒判断

### 真壁垒（稀缺且难复制）
1. **全球化组织能力**：跨文化、跨时区、跨区域的协同体系——长期沉淀，无法短期复制。
2. **本地化深度（L3-L4）**：本地团队 + 本地决策权 + 文化共鸣。
3. **B 端行业资源网络**：会员体系、招商渠道、合作生态——10 年+ 积累。
4. **标准化交付体系**：把非标服务标准化——规模化基础。
5. **多市场分散能力**：不依赖单一国家，抗地缘风险。

### 假壁垒
1. **"技术领先"**：AI 平权时代，纯技术领先窗口期 < 6 个月。
2. **"App Store 排名高"**：可能是买量刷出来的，可持续性差。
3. **"覆盖 100+ 国家"**：可能只是 App 上架，无真实运营。
4. **国内模式的简单复制**：抖音/拼多多的中国打法在海外不适用。

## 红旗识别
1. **海外营收 < 30% 但宣称"出海战略"**——只是国内业务的点缀。
2. **CAC > LTV/2 + 海外流量成本持续走高**——单位经济不成立。
3. **无专职海外团队（仅国内团队远程运营）**——组织能力不足。
4. **单一市场依赖 > 60%**（如只做美国/印度/印尼）——政策黑天鹅。
5. **海外产品与国内完全相同，无本地化适配**——L1 都没做到。
6. **核心团队近期频繁变动 + 海外负责人离职**——组织动荡。
7. **合规事件频发（GDPR/数据本地化违规）**——监管风险。
8. **过度依赖单一渠道（如只靠 TikTok 引流）**——渠道政策变化即崩塌。
9. **"覆盖多国"但每国营收 < 100 万美元**——广撒网无深耕。
10. **纯 AI 工具型产品 + 无独家内容/数据**——6 个月内可被复制。
