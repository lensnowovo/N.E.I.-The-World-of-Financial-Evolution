# 医疗健康（非生物医药：医疗服务/医疗器械/数字医疗）投资研究框架

## 这是什么
聚焦 **非生物医药部分**——连锁医疗服务（眼科/口腔/体检/医美）、医疗器械（高值耗材/影像/IVD）、数字医疗（AI 医疗/互联网医院/医疗 SaaS）的投资分析框架。适合 **PE/VC 医疗组、产业资本** 做赛道扫描与尽调。核心判断：**集采政策是最大变量；"自费 + 消费医疗 + 国产替代"三条主线穿越周期**。

## 行业研究框架（6 维度）

### 维度 1：政策环境与集采影响（中国特色首要维度）

**集采已进入深水区**（2020-2026 已完成 6 轮高值耗材国采）：
- 冠脉支架、骨科、人工晶体已集采完成——价格腰斩甚至更多。
- 电生理、神经介入、外周介入已扩面。
- **集采对产业链影响**：中间商/分销商被压缩；出厂价 vs 终端价差额被抹平——100 亿市场可能直接砍半。

**应对策略**：
1. **做医保外（自费）的品类**：医美、齿科、辅助生殖、高端眼科——不受集采影响。
2. **做能显著降医保支付成本的技术**：医保局视角倒推——能省钱的技术反而被支持。
3. **做集采后受益的国产替代品牌**：集采驱逐进口品牌，国产龙头份额提升。

### 维度 2：三个子赛道的差异化框架

#### A. 连锁医疗服务（眼科/口腔/体检/医美）

**共同逻辑**：单店盈利 → 区域复制 → 全国连锁 → 品牌壁垒。

**三个赛道差异对比**：

| 维度 | 眼科 | 口腔 | 体检 |
|---|---|---|---|
| **医生依赖度** | 最低（设备驱动） | **最高**（手动操作） | 中等 |
| **设备投入** | 最高（准分子/ICL 设备） | 中等 | 中高（影像设备） |
| **广宣费用率** | **最高**（获客激烈） | 较低 | 较低 |
| **客单价与复购** | 低频高客单 | 中频中客单 | 年度复购 |
| **国产替代空间** | 设备国产化 | 种植体国产化 | 设备国产化 |
| **龙头对标** | 爱尔眼科 | 通策医疗 | 美年健康 |
| **投资亮点** | 渗透率提升 + 并购整合 | "口腔消费化" + 种植牙集采放量 | 老龄化 + 数据变现 |

#### B. 医疗器械（高值耗材/影像/IVD）

**核心判断**："临床成功 ≠ 产品成功；产品成功 ≠ 市场成功"。

**三步验证**：
1. 临床价值大 → 必要非充分。
2. 产品成功 → 技术 + 注册证 + 合规。
3. 市场成功 → 医院渗透 + 医生教育 + 渠道 + 医保。

**长尾策略**：医疗器械市场细分、单品类规模不大——核心技术平台 → 多管线 → 长尾效应。

#### C. 数字医疗（AI 医疗/互联网医院/医疗 SaaS）

**核心判断**：AI 已深度渗透但 **临床落地信任壁垒** 是关键。

- **AI 影像**：肺结节/眼底/病理——已有三类证但商业化路径不清。
- **互联网医院**：复诊开方、慢病管理——依赖医保对接。
- **医疗 SaaS**：医院信息化、诊所管理——付费意愿低、定制化重。
- **AI 药物研发**：冲击传统 CRO，但商业模式仍在探索。

### 维度 3：医疗器械壁垒判断

**真壁垒**（按难度排序）：
1. **注册证（尤其三类）**：申请周期 2-5 年，临床数据要求高。
2. **临床数据与医生心智**：KOL 背书 + 临床文献 + 手术培训体系。
3. **渠道与服务网络**：经销商覆盖、售后维修、技术支持——10 年+ 积累。
4. **技术平台**：从单一技术 → 多管线平台（如酶切胶原蛋白技术 → 口腔/医美/运动医学）。
5. **成本优势**：中国制造的性价比——国产替代 + 出海双驱动。

**假壁垒**：
1. **"市场成功 ≠ 产品成功"**：靠销售能力把一个临床价值不大的产品卖大——一旦集采/政策变动即崩塌。
2. **单一产品线**：管线单一，无技术平台延展性。
3. **CDMO 模式做核心产品**：核心产品外包生产，无核心竞争力。

### 维度 4：国产替代与出海

**国产替代进度**（按国产化率）：
- **已完成替代**：监护仪、心电图机（国产化率 > 50%）。
- **快速追赶中**：医学影像（CT/MRI 国产率 20-30%）、消化内镜、电生理。
- **仍被海外主导**：手术机器人、高端 IVD、高端植入物。

**出海策略**：
- **并购海外经销商**：用中国性价比产品撬动北美/欧洲市场。
- **新兴市场优先**：东南亚、中东、拉美——门槛低、价格敏感、中国产品优势明显。
- **FDA/CE 认证**：出海门票。

## 壁垒判断（真假护城河）

### 真壁垒（医疗服务）
1. **品牌信任**：医疗消费是高信任决策，口碑积累 10 年+。
2. **医生团队**：顶级医生稀缺（口腔/医美尤其）。
3. **选址卡位**：核心商圈/医院周边优质点位——先到先得。
4. **总部中后台**：供应链、培训、IT 系统——连锁化基础。

### 真壁垒（医疗器械）
1. **三类注册证 + 临床数据**。
2. **KOL + 医生培训生态**。
3. **技术平台延展性**（单技术 → 多管线）。
4. **全球渠道网络**。

## 红旗识别
1. **集采核心品种占营收 > 50% 且无替代产品线**——政策黑天鹅。
2. **单一产品依赖 > 70%**——管线风险。
3. **应收账款激增（医院回款慢）+ 现金流转负**——资金链风险。
4. **管理层用 CDMO 代工核心产品**——核心竞争力外包。
5. **广宣费用率持续 > 15% 但新客增长停滞**——品牌势能衰退。
6. **医生流失率 > 30%/年**（口腔/医美）——核心资产流失。
7. **海外营收占比 < 5% 却宣称"出海战略"**——纯概念。
8. **飞检/合规处罚频发**——质量体系缺陷。
9. **高溢价并购产生大额商誉（> 净资产 50%）**——商誉减值地雷。
10. **管理层在 IPO 前大规模套现**——利益取向存疑。
