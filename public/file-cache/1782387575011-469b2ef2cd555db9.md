# 生命周期三步法：如何快速了解一个行业

## 这是什么
一套以"产业生命周期"为主线的行业分析方法论，通过先判断行业所处阶段，再决定分析重点，避免无脑套模型。适合投资经理、咨询顾问、创业者在 1–2 周内对一个陌生行业建立系统认知。

## 方法论框架

### 第一步：确认行业所处生命周期阶段（1–2 天）
- 用 **渗透率** 替代传统的营收时间曲线作为判断标准（更前瞻）。
- 判断行业处于四阶段中的哪一段：导入期 / 成长期 / 成熟期 / 衰退期。
- 辅助信号：企业数量、市场规模增速、产品价格走势、平均盈利水平、竞争格局集中度、投资热度。

### 第二步：按阶段敲定分析重点（避免撒胡椒面）
不同阶段对应的关键问题不同：

| 行业阶段 | 核心问题 | 关键分析维度 |
|---|---|---|
| **导入期** | 商业模式是否成立？需求是否为真？ | 商业模式可行性、需求验证、盈利可持续性、技术成熟度 |
| **成长期** | 市场是否足够大？增速有多久？ | 市场规模 TAM/SAM/SOM、未来 3–5 年 CAGR、渗透率提升斜率 |
| **成熟期** | 地位是否稳固？能否抵御替代？ | 护城河、竞争格局 CR4/CR10、产业链利润分配、集中度变化 |
| **衰退期** | 是终结还是焕发新生？ | 替代品威胁分析、第二增长曲线、新技术/新模式/新市场 |

### 第三步：按图索骥填充信息（3–5 天）
对每个重点维度依次回答五个核心问题：
1. **行业概况**：规模、增速、价值链分工、利润分配、驱动因素（技术/政策/需求）。
2. **竞争格局**：集中度、波特五力（潜在进入者、替代品、供需方议价能力、现有竞争者）。
3. **商业模式**：盈利模式、关键资源、护城河类型（品牌/技术/渠道/网络效应/转换成本/规模经济/特许权利）。
4. **监管与政策**：政策导向（鼓励/限制）、准入门槛（资质、牌照）、合规要求。
5. **风险与趋势**：技术迭代风险、政策变动风险、颠覆性变化、机会点。

### 第四步：验证与结构化输出（1–2 天）
- **交叉验证**：对比专家访谈 vs 数据报告，识别矛盾点并深挖原因。
- **输出逻辑闭环**：用金字塔原理（核心观点→分论点→数据/案例支撑）。
- **可视化**：绘制产业链图谱 + 商业模式画布。

## 红旗识别（这些信号说明分析可能出问题）
1. **用导入期的框架去分析成熟期公司**（比如对一家成熟制造业龙头还在算 TAM，却忽略护城河和竞争格局）。
2. **渗透率数据来源单一**：仅用一份券商报告，未交叉验证，可能严重高估/低估。
3. **把"产品好"误认为护城河**：产品好不等于壁垒，要看是否有结构性优势（专利、网络效应、转换成本、规模）。
4. **忽略替代品威胁**：只看同赛道竞争对手，忽视跨赛道替代（如新能源车对燃油车的替代、AI 对人工服务的替代）。
5. **生命周期判断错位**：把"成长期末段"误判为"成熟期"，导致过度关注护城河而错失最后的规模红利。
6. **政策维度缺失**：在中国市场，尤其是教育、医疗、金融、平台经济，缺政策分析等于缺半边天。
7. **专家访谈样本偏差**：只访谈行业内的乐观派（创始人/销售），没有访谈客户、供应商、离职员工。
8. **未做动态推演**：只看静态快照，没有推演"下一阶段会发生什么"。
