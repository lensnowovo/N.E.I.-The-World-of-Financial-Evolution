# 一级市场行业研究端到端工作流：从陌生赛道到 IC 立项报告

## 这是什么

把 N.E.I. 上的行业框架 + 投研能力 + 交付工具**串联成一个完整工作流**——拿到一个陌生赛道，从 0 到完成一份 IC 级行研报告，每一步告诉你该用哪个 Skill、看什么、怎么验证。适合 PE/VC/FA 投资经理、分析师、IR。

**核心理念**：行业研究不是"搜资料 → 写报告"，而是"判断阶段 → 定向收集 → 交叉验证 → 结构化输出"的系统工程。用错框架看错行业 = 浪费一周写废报告。

## 工作流全貌（8 步）

```
第 0 步：判断行业生命周期阶段
    ↓
第 1 步：政策环境扫描（中国特色必做）
    ↓
第 2 步：技术前沿与竞争管线追踪
    ↓
第 3 步：产业链拆解 + 市场规模测算
    ↓
第 4 步：行业专属深度分析（调用对应行业框架）
    ↓
第 5 步：专利壁垒评估（硬科技必做）
    ↓
第 6 步：专家验证 + 定量交叉验证
    ↓
第 7 步：输出 IC 立项报告
```

---

## 第 0 步：判断行业生命周期阶段（30 分钟）

**调用 Skill**：[生命周期三步法 #67](/posts/67)

**做什么**：
1. 查渗透率，判断行业处于四阶段中的哪段（导入期 / 成长期 / 成熟期 / 衰退期）
2. 按阶段敲定后续分析重点（不是所有维度都要看）

**关键判断**：
| 阶段 | 核心问题 | 后续重点 |
|---|---|---|
| 导入期 | 需求是否为真？商业模式能否成立？ | 技术可行性、需求验证 |
| 成长期 | 市场多大？增速能持续多久？ | TAM/SAM/SOM、渗透率斜率 |
| 成熟期 | 地位能否稳固？能否抵御替代？ | 护城河、竞争格局 CR4 |
| 衰退期 | 有没有第二增长曲线？ | 替代品分析、转型可能 |

**红旗**：渗透率数据来源单一 / 把"产品好"当护城河 / 忽略跨赛道替代品。

**产出**：一行结论——"XX 行业处于 [阶段]，分析重点 = [XX]"。

---

## 第 1 步：政策环境扫描（1-2 小时）

**调用 Skill**：[政策检索方法论 #89](/posts/89)

**做什么**：
1. 4 层信息源搜索（国家级 → 部委 → 专项 → 地方）
2. 判断政策信号方向（鼓励 / 中性 / 限制 / 整顿）
3. 标注政策信号重要性（🔴中央文件 > 🟠国务院 > 🟡部委 > 🟢讲话）

**中国特色必查**：
- 产业政策（扶持补贴 / 准入限制 / 税收优惠）
- 监管态度（鼓励创新 / 趋严 / 专项整顿）
- 政府引导基金投向（参考 [应对政府引导基金方法论 #97](/posts/97)）
- 出口管制 / 实体清单影响（硬科技必查）

**红旗**：只看媒体报道不查原文 / 漏掉征求意见稿（政策演变前兆）/ 引用了已废止政策。

**产出**：政策矩阵——"政策方向 = [鼓励/中性/限制]，关键文件 = [文号+日期]，信号强度 = [🔴🟠🟡🟢]"。

---

## 第 2 步：技术前沿与竞争管线追踪（2-3 小时）

**调用 Skill**：[论文与前沿动态检索 #90](/posts/90)

**做什么**：
1. 找到这个行业的技术顶会/顶刊（AI→NeurIPS、医药→CNS、半导体→IEDM）
2. 追踪 3-5 个核心 PI/实验室的最新发表
3. 设置 Alert（arXiv 日级 / Scholar 周级）
4. 判断近期突破是"真突破"还是"incremental"（7 维度判断法）

**硬科技专属补充**：
- 生物医药：查 Clinicaltrials.gov 竞品管线布局 → 参考 [创新药/CGT 框架 #79](/posts/79)
- 半导体：查技术路线（制程/Chiplet/第三代半导体）→ 参考 [半导体框架 #87](/posts/87)

**红旗**：只会百度搜论文 / 不分顶会顶刊 / 被"重大突破"新闻误导（要看预印本+被引+独立验证）。

**产出**：技术地图——"主流路线 = [XX]，近期突破 = [XX]，竞争管线 = [3-5 家]"。

---

## 第 3 步：产业链拆解 + 市场规模测算（3-4 小时）

**调用 Skill**：
- [3 面 10 点行研框架 #68](/posts/68) — 产业链 + 价值链 + 竞争格局
- [TAM/SAM/SOM 测算 SOP #70](/posts/70) — 市场规模

**做什么**：
1. 画产业链全景图（上游原材料 → 中游制造 → 下游应用）
2. 标注各环节利润分配（谁在赚大头、谁在苦力）
3. 测算市场规模：自上而下（行业数据 → 切份额）+ 自下而上（用户数 × ARPU）
4. 竞争格局分析：CR4/CR10 集中度 + 波特五力

**判断标准**：
- 市场规模千亿以上 + 增速 20-30% = 好赛道
- CR4 < 30% 且无集中趋势 = 原子型市场，难出大公司
- 产业链利润严重向一端倾斜 = 另一端可能是苦力

**红旗**：TAM 定义过宽 / 只做自上而下不做验证 / 增速假设 >50% 无逻辑。

**产出**：产业链图 + 市场规模表（乐观/中性/悲观三档）。

**红旗**：用通用框架套所有行业（撒胡椒面）/ 忽略行业特有的核心指标（如半导体不看良率、创新药不看临床阶段）。

**产出**：行业深度分析（5-10 页），含核心指标表 + 壁垒判断 + 红旗清单。

## 第 5 步：专利壁垒评估（硬科技必做，2-3 小时）

**调用 Skill**：[专利检索与分析 #91](/posts/91)

**做什么**：
1. 检索目标公司 + 竞品的核心专利
2. FTO 自由实施评估 5 步法（是否侵犯他人专利）
3. 同族专利分析（全球布局意图）
4. 专利到期追踪（核心专利何时过期 → 仿制窗口）

**红旗**：只会搜公司名 / 不分发明 vs 实用新型 / 凑数专利多核心专利少 / 无 FTO 评估。

**产出**：专利壁垒评分（强/中/弱）+ FTO 风险等级。

---

## 第 6 步：专家验证 + 定量交叉验证（1-2 天）

### 6a. 专家访谈
**调用 Skill**：[专家访谈方法论 #92](/posts/92)

**做什么**：
1. 筛选 3-5 位专家（离职 1-6 月中层最佳）
2. 前 15 分钟试金石（验证专业度）
3. 5 段式大纲 + 定量数据层层递进
4. 交叉验证（单一专家不能定结论）

### 6b. 招投标/订单验证（To B / To G 项目必做）
**调用 Skill**：[招投标与订单验证 #93](/posts/93)

**做什么**：
1. 拆解目标公司前 5 大客户
2. 反查招投标平台（采购意向 + 中标 + 合同）
3. 比对宣称收入 vs 公开订单（4 分类：✅吻合 / ⚠️部分 / 🔴严重不符 / 🚫查不到）
4. 竞争格局分析（集中度 / 价格趋势 / 新进入者）

**红旗**：宣称收入 vs 招投标严重不符 / 多家关联公司投标（陪标）/ 单一客户 >70%。

**产出**：专家访谈纪要（3-5 份）+ 订单验证报告。

---

## 第 7 步：输出 IC 立项报告

**调用 Skill**：
- [3 面 10 点行研框架 #68](/posts/68) — 报告章节结构
- [2026 VC 8 大新标准 #69](/posts/69) — 打分维度

**报告结构**（直接对应 3 面 10 点）：

```
1. 执行摘要（1 页，核心结论先行）
2. 行业概况（规模/增速/生命周期阶段/驱动因素）
3. 政策环境（方向/信号/风险）
4. 产业链分析（全景图 + 利润分配）
5. 竞争格局（CR4/CR10 + 波特五力）
6. 技术趋势（主流路线/近期突破/替代风险）
7. 目标公司分析（业务/团队/财务/专利/订单验证）
8. 风险与红旗（≥10 条，分级）
9. 投资建议（推荐/观望/不推荐 + 估值区间）
```

**附 2026 VC 8 大新标准自检**：
- ✅ 算力数据合规
- ✅ 供应链安全
- ✅ ESG
- ✅ 国产替代率
- ✅ 专利壁垒
- ✅ 客户粘性（复购/续约）
- ✅ 退出确定性
- ✅ 政策风险敞口

## 红旗识别（工作流级别的警示）

1. **跳过第 0 步直接搜资料**——阶段判断错，后面全错。
2. **政策维度缺失**——中国市场政策是最大变量（教育/医疗/金融科技/平台经济教训）。
3. **用通用框架套硬科技**——半导体不看良率、创新药不看临床阶段 = 外行。
4. **不做交叉验证**——只看研报不访谈专家、只听 CEO 说不看招投标 = 信息茧房。
5. **第 4 步行业框架选错**——把消费连锁框架套在半导体上 = 浪费时间。
6. **专利评估缺失**——硬科技投资不做 FTO = 定时炸弹。
7. **专家样本偏差**——只访谈乐观派（创始人/销售），不访谈客户/供应商/离职员工。
8. **报告没有红旗章节**——只写机会不写风险的报告 = 不负责任。
