# 科学家创业 / 深科技投资研究框架（跨领域通用版）

## 这是什么
覆盖 AI4S / 合成生物 / 量子科技 / 脑科学 / 光电芯片 / 新材料等**所有"科学家创业"赛道**的通用投资框架。核心命题不是某个具体技术，而是 **"如何评估学术 IP 的可商业化"**——这是跨所有深科技赛道的共同方法论。适合 PE/VC 早期组、孵化器、高校技术转移办公室、科技成果转化基金。**2025-2026 年行业认知的关键迭代：从"迷信教授光环"到"科学家 + 资深产业界人士 + 资本"组合配置。**

## 行业研究框架（6 维度）

### 维度 1：创始人/团队评估（最核心维度）

> 投资人最关注的要素。2021 年"抢教授"热潮退去后，行业认知大幅迭代。

**获融资团队的 4 个共性特征**：

| 特征 | 含义 | 代表案例 |
|---|---|---|
| **① 创始人有产业经验** | 创业/转化经验，懂商业化 | 血霁生物朱芳芳（曾任职 WI Harper Group 投资部） |
| **② "科学家 + 产业界"组合** | 教授专注研发 + 资深管理者运营 | 正序生物（陈佳教授 + 牟晓盾 CEO 前药明/默沙东） |
| **③ 全球首创技术** | 原始创新，不是 me-too | 阿泰克（复旦 Nature 论文 ATTEC 技术） |
| **④ 破釜沉舟决心** | 全职创业，不留退路 | 波达医疗许凯亮（对赌协议 + 全职） |

**"科学家 + 资深产业界 + 资本"= 顶级配置**，这是行业共识。

**被淘汰团队的共性问题**：
1. 教授中途动摇，回归学术。
2. 兼职创业，关键期缺位（融资/谈判/BD 时找不到人）。
3. 只懂技术创新，忽视市场逻辑与产业竞争。
4. "如果不成功就回学校"——留退路 = 缺乏坚持。

### 维度 2：学术 IP 的商业化评估

**5 层评估框架**（从 Paper 到 Product）：

| 层级 | 问题 | 评估方式 |
|---|---|---|
| **L1 科学有效性** | 机制是否真实？ | 顶刊论文 + 独立重复验证 |
| **L2 技术可行性** | 实验室能否复现稳定？ | 亲自复现数据（投资人要求） |
| **L3 工程化** | 能否放大生产？良率如何？ | 中试数据、批次稳定性 |
| **L4 商业化** | 下游客户是否愿意付费？ | 真实订单、意向书（非口头） |
| **L5 规模化** | 成本曲线能否下降？ | 单位成本随规模的学习曲线 |

**关键原则**：**"从 Paper 到 Product 是两个世界"**——学术创新 ≠ 商业成功。投资人要求复现试验数据（如合成生物的菌株表达量），以合理评估成本。

### 维度 3：选品/选题策略（深科技的核心战略决策）

> 合成生物领域的经验对所有深科技赛道有普适意义：**"选品决定生死"**。

**两种路径对比**：

| 路径 | 特征 | 优势 | 风险 |
|---|---|---|---|
| **赌新分子/新技术** | 高附加值，首创性 | 溢价高，先发优势 | 下游需求未爆、准入监管 |
| **卷旧分子/成熟产品** | 大宗化学品/成熟应用 | 市场确定，天花板高 | 重资产、与巨头拼成本 |

**"命题作文"原则**：市场需求明确或有渠道拿到订单时，才投入量产——而非"有了技术找市场"。

**合成生物的"平衡点"经验**（峰瑞资本马睿）：选品在"量产规模 + 附加值 + 市场空间"三者之间找平衡。5000-10000 吨规模 + 较高附加值 + 不错市场空间 = 踏实选择。

### 维度 4：商业化节奏与里程碑

**财务投资人的典型期待周期**（峰瑞资本）：
- 第 1 年：融资。
- 第 2 年：建厂/建实验室。
- 第 3 年：产生收入。
- 第 4 年：做出规模化业绩。

**实际节奏通常慢于预期**——商业化拓展难以一蹴而就。深科技公司基本都经历了 10 年+ 积累才成为细分龙头。

**里程碑体系**（分阶段）：
| 阶段 | 关键里程碑 | 融资节点 |
|---|---|---|
| **种子** | 科学验证 + 团队组建 | 天使/孵化 |
| **A 轮** | 实验室复现 + 初步工程化 | 早期 VC |
| **B 轮** | 中试 + 首个商业订单 | 成长期 VC |
| **C 轮+** | 量产 + 规模化收入 | PE/产业资本/IPO |

### 维度 5：中国特色——政策红利与资本结构

**2024-2026 政策密集出台**：
- "生物制造"被定义为战略性新兴行业（与商业航天、低空经济并列）。
- "新质生产力"核心赛道——AI4S/合成生物/量子/脑科学。
- 国央企支持任务：招商局/国投/国药/中粮等领了战略新兴产业支持任务。

**资本结构变化**（深科技早期）：
1. **国家队/国央企基金**：看重战略价值，估值宽容度高。
2. **地方政府引导基金**：有招商需求（"搬工厂换投资"），但单个项目上限约 1 亿元 + 终身追责。
3. **市场化财务投资人**：要求估值与商业化进程匹配，参考上市公司 PE/PS。
4. **产业方 CVC**：用资源（产能/渠道/品牌）换估值。

**地方基金的限制**：公司搬至 A 城市即不可逆，A 政府能给的有限——"不足以救命"。

### 维度 6：知识产权与竞争壁垒

**深科技 IP 的特殊性**：
- **不像创新药专利保护那样成熟**——生物技术/材料配方"像窗户纸"，有了方向别人花时间也能做出来。
- **不像半导体高端制造壁垒显著**——工艺 know-how 沉淀更深。

**壁垒构建策略**：
1. **在技术领先的时间差里，建立规模和渠道先发优势**——而非纯靠专利。
2. **管线/技术平台化**——单技术 → 多管线 → 平台价值。
3. **数据飞轮**（AI4S 尤其）——数据越多，模型越强。
4. **BD 交易**——License-out 出海授权，类似创新药"首付款 + 里程款 + 销售提成"。

## 行业核心指标

| 指标 | 健康阈值 | 说明 |
|---|---|---|
| **创始团队配置** | 科学家 + 产业界 + 资本 | 顶级配置 |
| **创始人投入度** | 全职（非兼职） | 破釜沉舟 |
| **学术 IP 层级** | L3-L5（工程化以上） | 纯 L1-L2 太早 |
| **顶刊论文数** | Nature/Science/Cell ≥ 1 | 科学有效性背书 |
| **独立复现验证** | 已完成 | 投资人必查 |
| **中试/量产数据** | 批次稳定性 > 90% | 工程化能力 |
| **真实付费订单** | 有（非口头意向） | 商业化证据 |
| **单位成本学习曲线** | 随规模下降 | 规模化潜力 |
| **研发费用率** | > 30%（早期） | 技术驱动 |
| **现金 runway** | > 18 个月 | 深科技长周期 |
| **管线数量** | ≥ 2（避免单一管线） | 风险分散 |
| **BD/License-out 交易** | 有（加分项） | 出海能力 |

## 壁垒判断

### 真壁垒（深科技）
1. **底层原创技术 + 顶刊背书**：Nature/Science/Cell 论文 = 科学有效性的最强背书。
2. **技术平台延展性**：单技术 → 多管线（如酶切胶原蛋白技术 → 口腔/医美/运动医学）。
3. **数据飞轮**（AI4S）：用户/实验数据回流到模型优化——先发优势。
4. **规模 + 渠道先发**：在技术领先时间窗内建立。
5. **人才稀缺性**：顶尖科学家 + 资深产业界组合——"创业合伙人是稀缺资源"。

### 假壁垒（警惕）
1. **"教授光环"**：名校教授 ≠ 成功创业者。
2. **"顶刊论文 = 商业价值"**：学术创新 ≠ 商业成功。
3. **"技术领先"无规模/渠道支撑**：技术外溢不可避免，时间差是唯一优势。
4. **"ESG/绿色概念"**：客户最终只关心降本增收，不为概念买单。
5. **兼职创业的"资源"**：高校资源 ≠ 商业化能力。

## 红旗识别
1. **教授兼职创业 + 关键期缺位**——最大失败原因。
2. **"如果不成功就回学校"心态**——缺乏破釜沉舟决心。
3. **无产业经验的纯学术团队**——市场化短板致命。
4. **学术 IP 停在 L1-L2（纯实验室）**——距离商业化太远。
5. **拒绝复现试验数据**——数据存疑。
6. **单一管线/单一产品，无平台延展性**——一损俱损。
7. **"卷旧分子"但与巨头拼成本无优势**——重资产陷阱。
8. **"赌新分子"但下游需求未爆 + 价格已卷**——麦角硫因案例（15 万→3 万/公斤）。
9. **To VC 故事驱动（而非 To Customer）**——技术故事万能论。
10. **地方政府基金主导但商业化路径不清**——"搬工厂换投资"不可持续。
11. **管线/应用场景"藏起来"但实际是没找到**——方向不清。
12. **核心科学家近期离职/跳槽**——知识资产流失。
