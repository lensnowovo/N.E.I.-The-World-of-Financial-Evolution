# 新能源 / 气候（光伏/储能/锂电）投资研究框架

## 这是什么
新能源产业链（光伏/储能/锂电池/风电/电网）的投资分析框架。适合 **PE/VC/二级市场投资人** 在"双碳"政策主题和产业链转移背景下做赛道判断。新能源是典型的**政策驱动 + 技术迭代 + 产能周期**三重叠加行业，核心矛盾是：**产业趋势明确，但产能过剩与估值消化是常态**，投资逻辑从"规模扩张"转向"质量优先"。

## 行业研究框架（6 维度）

### 维度 1：产业链全景与价值分配

```
光伏产业链                  锂电产业链
├── 硅料（上游，化工属性）    ├── 矿产资源（锂/钴/镍，资源属性）
├── 硅片                    ├── 正极/负极/电解液/隔膜（材料）
├── 电池片                  ├── 电芯制造
├── 组件                    ├── Pack/模组
├── 逆变器                  └── 应用（动力电池/储能）
├── 支架/辅材
└── 电站运营（下游央国企）

储能产业链
├── 电化学储能（锂电/钠电/液流）
├── 机械储能（抽水/压缩空气）
├── 电磁储能（超级电容）
└── 氢储能
```

**利润分配判断**：产能紧张环节吃超额利润，产能过剩环节利润被挤压。当前格局：
- **硅料**：扩产周期长（1.5-2 年），化工属性强，周期性明显。
- **隔膜**：扩产谨慎、产能释放周期长、竞争格局清晰——产业链中相对优质环节。
- **电池**：龙头集中度高，CR3 > 60%。

### 维度 2：政策驱动与碳中和目标
- **国家目标**：2030 碳达峰、2060 碳中和——长期产业趋势确定性。
- **短期政策波动**：补贴退坡、配储政策、绿电交易、特高压规划。
- **中国特色**：央国企能源集团是下游主力买家，通过招投标采购——对组件价格影响力强。
- **海外政策**：美国 IRA 法案、欧盟碳边境税、双反调查——影响出海逻辑。

### 维度 3：成本曲线与技术迭代
- **LCOE（平准化度电成本）**：核心量化指标。
  - 光伏：0.15-0.24 元/kWh（2024）；配储后综合 0.26-0.38 元/kWh。
  - 中国新增光储项目 < 0.3 元/kWh，低于欧美（0.4-0.6 元/kWh）。
- **技术路线迭代**：
  - 光伏：PERC → TOPCon → HJT → 钙钛矿叠层。
  - 锂电：磷酸铁锂 → 三元 → 钠离子 → 固态。
  - 储能：锂电为主，液流/压缩空气为补充。
- **学习曲线**：累计装机翻倍，成本下降 X%（光伏约 20%）。

### 维度 4：产能周期与供需格局
- **产能利用率**：> 70% 健康；< 60% 严重过剩。
- **库存周期**：硅料/电池片库存波动直接影响价格。
- **扩产周期差异**：硅料（1.5-2 年）vs 电池片（6-9 个月）vs 组件（3-6 个月）——扩产慢的环节周期性强。
- **行业 CR3/CR5**：集中度提升期，龙头受益；分散期，价格战。

### 维度 5：技术壁垒与差异化
- **设备壁垒**：核心设备国产化率、定制化程度。
- **工艺壁垒**：良率、转换效率、循环寿命——精密制造 know-how。
- **材料壁垒**：上游矿产资源（锂/钴/镍）议价能力。
- **认证壁垒**：车规级认证、UL/CE 安全认证。

### 维度 6：出海与产业链转移
- **中国优势**：2025 年全球 80% 光伏组件、2/3 动力电池来自中国。
- **贸易壁垒升级**：美国对华关税、欧盟碳关税、印度 PLI 计划。
- **应对策略**：海外建厂（宁德德国/匈牙利、隆基越南）、技术授权、合资。
- **估值重估机会**：出海规模→市场空间；全球价值链地位→盈利质量；出海壁垒→竞争格局。

## 壁垒判断

### 真壁垒
1. **规模效应 + 成本曲线领先**：龙头单位成本比二三线低 15-30%，周期底部仍能盈利。
2. **工艺 know-how**：良率每提升 1% = 净利率提升 2-3 个百分点。
3. **上游资源控制**：锂矿/钴矿所有权，资源品涨价时议价能力。
4. **客户长单**：车企/电站央国企的长期供货协议（5-10 年）锁定收入。
5. **技术专利**：核心材料/结构专利（如宁德麒麟电池、钙钛矿叠层）。

### 假壁垒
1. **"已扩产"≠ 已投产**：规划产能 vs 实际投产差异大。
2. **低价中标**：储能 EPC 低价中标但质量缩水——口碑恶化。
3. **政策补贴依赖**：补贴退坡后盈利能力消失。

## 红旗识别
1. **产能利用率 < 60% 仍在对外宣称扩产**——产能过剩加剧。
2. **毛利率持续下滑，跌破 15%**——价格战白热化。
3. **应收账款激增 + 客户集中于央国企**——议价权弱、回款慢。
4. **技术路线押错（如死守 PERC 不转 TOPCon）**——被迭代淘汰。
5. **海外贸易摩擦升级 + 高度依赖单一市场**——出海受阻。
6. **核心专利诉讼缠身**——海外市场禁令风险。
7. **库存积压 > 6 个月销量**——周期顶部信号。
8. **管理层在大规模减持**——产业资本离场信号。
9. **补贴占净利润比 > 50%**——商业模式脆弱。
10. **扩产周期错配**（在周期顶部扩产）——投产即过剩。
