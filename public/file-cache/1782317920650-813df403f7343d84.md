# 买家清单：给一个项目列出潜在战略 + 财务买家

这是 Anthropic 官方的 买家清单（Buyer List） skill。卖项目前列出所有潜在买家：

- 战略买家：同业 / 上下游 / 跨界，按协同效应排序

- 财务买家：PE 基金，按行业偏好和基金规模筛选

- 每个买家标注：历史交易、当前持仓、接触难度

- 优先级排序：先碰谁、后碰谁

适合：FA 做卖方流程的起点。买家清单的质量直接决定交易能不能成。
