# 半导体 / 芯片投资研究框架

## 这是什么

半导体/芯片产业（IC 设计 / 制造 / 设备 / 材料 / 封测 / EDA）的投资分析框架。这是**国家战略 + 周期 + 技术迭代**三条线叠加的赛道——既是"卡脖子"国家安全的必投方向，也是典型的周期性行业，同时正经历 AI / 先进制程 / 第三代半导体等技术迭代。核心判断：**国产替代主线（从"能用"到"好用"）+ 技术迭代主线（AI 算力 / Chiplet / 新材料）**，两条线决定不同环节的投资优先级。**大基金三期 3440 亿元 + MATCH 法案（美国）**重塑全球格局，设备与材料是当前国产化率最低、成长空间最大的环节。适合 PE/VC 硬科技组、产业资本、国家大基金、地方政府半导体基金。

## 行业研究框架（7 维度）

### 维度 1：两条投资主线（核心判断）

> 半导体不是"一个行业"，而是两条主线叠加——判断项目属于哪条线，决定估值逻辑和退出路径。

| 主线 | 驱动力 | 投资逻辑 | 代表环节 |
|---|---|---|---|
| **国产替代** | 国家安全 + 卡脖子 + 供应链自主 | 政策红利 + 客户认证 + 从"能用"到"好用" | 设备、材料、EDA、先进制程制造 |
| **技术迭代** | AI 算力 / 先进封装 / 第三代半导体 / RISC-V | 技术领先 + 新场景需求爆发 | GPU/AI 芯片、HBM、Chiplet、SiC/GaN |

**关键认知**：
- **国产替代线**：政策/资本驱动，客户认证周期是核心门槛，"能用"→"好用"是价值跃迁点。
- **技术迭代线**：市场需求驱动，技术领先度 + 量产能力是核心，估值弹性大但风险高。
- **最佳项目**：同时踩中两条线——既有国产替代的确定需求，又处于技术迭代的关键节点。

### 维度 2：产业链全景与国产化率（核心价值图谱）

```
IC 设计                    国产化率 ~20-30%（逻辑芯片更低）
├── CPU / GPU / AI 芯片     ← 技术壁垒最高，国产化率 < 10%
├── 存储（DRAM/NAND）       ← 重资产，长江存储/长鑫存储
├── 模拟芯片                ← 国产化率较高（30%+），分散市场
├── 射频 / 功率器件          ← 第三代半导体机会
└── EDA 工具                ← 国产化率 < 15%，卡脖子最严重之一

IC 制造                    国产化率 ~20%
├── 先进制程（7nm 及以下）  ← 受制裁影响，设备受限
├── 成熟制程（28nm+）       ← 国产替代主战场，产能扩张
└── 特色工艺（BCD/SOI）     ← 特色代工有差异化

半导体设备 ⭐核心机会        国产化率 ~21%（2024，整体偏低）
├── 光刻机                  ← 国产化率 < 5%，EUV 完全空白
├── 刻蚀设备                ← 国产化率 ~20-35%（中微/北方华创）
├── 薄膜沉积（PVD/CVD/ALD） ← 国产化率 ~10-30%
├── 离子注入                ← 国产化率 ~10-15%
├── CMP（化学机械抛光）     ← 国产化率 ~20-30%
├── 量测设备                ← 国产化率 ~5-15%
├── 涂胶显影                ← 国产化率 < 10%（被东京电子垄断）
└── 热处理 / 清洗           ← 国产化率 30%+（较高）

半导体材料 ⭐核心机会        国产化率 ~20-30%
├── 硅片（12 寸大硅片）     ← 国产化率 ~20%
├── 光刻胶                  ← 国产化率 < 15%（高端空白）
├── 电子特气                ← 国产化率 ~30%
├── 湿电子化学品            ← 国产化率 ~30%
├── CMP 抛光液/垫           ← 国产化率 ~20-30%
├── 掩膜版                  ← 国产化率 ~15%
└── 靶材                    ← 国产化率 ~40%（较高）

封测（OSAT）               国产化率最高（> 60%）
├── 传统封装                ← 成熟，长电/通富/华天
├── 先进封装（CoWoS/2.5D/3D）← AI 芯片刚需，国产起步
└── Chiplet                ← 技术迭代主线，连接设计与封测
```

**价值分配判断**：
- **设备 + 材料是当前最佳投资窗口**——国产化率最低（20% 左右）、成长空间最大、政策资金最集中。
- **设计端**：竞争激烈，只有"卡脖子设计"（CPU/GPU/EDA）或"新场景设计"（AI 芯片/车规）有高溢价。
- **制造端**：重资产、长周期，适合大基金/产业资本，非典型 VC 标的。
- **封测端**：成熟行业，先进封装是唯一增量亮点。

### 维度 3：半导体设备细分赛道投资优先级（⭐ 重点扩展）

> 设备是半导体投资的核心机会——国产化率最低 + 客户粘性最强 + 技术壁垒最高。

| 设备类别 | 国产化率 | 技术壁垒 | 投资优先级 | 国产代表 | 关键判断 |
|---|---|---|---|---|---|
| **光刻机** | < 5% | ★★★★★ | ★★★（最难） | 上海微电子 | EUV 完全空白，DUV 国产突破中，长周期 |
| **刻蚀** | 20-35% | ★★★★ | ★★★★★（最优） | 中微、北方华创 | 已进入先进制程，国产化率提升最快 |
| **薄膜沉积（CVD/ALD）** | 10-30% | ★★★★ | ★★★★★ | 拓荆、北方华创 | ALD 是先进制程刚需，空间大 |
| **CMP** | 20-30% | ★★★ | ★★★★ | 华海清科 | 已实现 28nm 国产替代，向先进制程渗透 |
| **离子注入** | 10-15% | ★★★ | ★★★★ | 凯世通（万业） | 大束流/高能国产化空间大 |
| **量测设备** | 5-15% | ★★★★ | ★★★★★ | 睿励、中科飞测 | 国产化率极低 + 技术壁垒高 = 蓝海 |
| **涂胶显影** | < 10% | ★★★★ | ★★★★ | 芯源微 | 东京电子全球垄断，国产替代空间巨大 |
| **热处理/清洗** | 30%+ | ★★ | ★★★ | 北方华创、盛美 | 国产化率高，增量放缓 |

**设备投资核心逻辑**：
1. **看客户认证进度**——头部晶圆厂（中芯/华虹/长江存储/长鑫）的产线验证是生死线。
2. **看"从能用到好用"的跨越**——突破验证门槛只是第一步，良率/稳定性达到甚至超越海外产品才是壁垒。
3. **看合同负债/预收款周转天数**——这是设备公司订单质量的核心验证指标。
4. **看产品线扩展能力**——单一设备难成大公司，平台化（多品类）是长期价值。

### 维度 4：客户认证周期与"能用→好用"跃迁

> 半导体设备/材料的客户认证是全行业最严苛的——一旦进入产线，粘性极强。

**认证流程（以设备为例）**：
```
研发 → 样机 → 客户实验室验证 → 产线小批量试 → 产线批量验证 → 正式纳入采购 → 持续供货
（1-2 年）              （6-12 月）     （6-12 月）       （6-12 月）        （长期）
```

**"能用→好用"的价值跃迁**：
- **"能用"阶段**：产品通过基本验证，进入产线试运行——**估值的第一波跃迁**。
- **"好用"阶段**：良率/稳定性/综合成本达到或超越海外竞品——**估值的第二波跃迁**（真正确立壁垒）。
- 很多公司卡在"能用"无法跨越到"好用"——技术/服务/迭代能力是分水岭。

**客户粘性的"护城河"**：
- 一旦设备/材料正式纳入产线，更换成本极高（需重新验证、影响良率、产线停产风险）。
- 因此**先发优势 + 头部客户绑定 = 极强的客户粘性**。

### 维度 5：政策与资本格局

**国家大基金**：
- **一期（2014）**：1387 亿元，投制造为主。
- **二期（2019）**：2042 亿元，投设计 + 设备。
- **三期（2023）**：**3440 亿元**（史上最大），重点投**设备 + 材料 + EDA + 先进存储（HBM）**——这是当前最重要的资金风向标。

**MATCH 法案（美国，2024-2025）**：
- 美国主导的"半导体多边出口管制联盟"——日本/荷兰/韩国等加入。
- 进一步限制先进设备（EUV 光刻、先进刻蚀、ALD 等）对华出口。
- **影响**：短期加大国产替代紧迫性（利好国产设备），长期可能影响先进制程追赶节奏。

**地方政府基金**：
- 各地半导体基金（上海/合肥/武汉/深圳等）+ 产业园模式。
- 注意："搬公司换投资"模式不可持续，关注真实产业基础。

### 维度 6：技术迭代主线（AI / Chiplet / 第三代半导体）

**AI 算力驱动**：
- GPU / AI 芯片需求爆发（英伟达 H100/B200 → 国产昇腾/寒武纪/壁仞）。
- **HBM（高带宽存储）**：AI 训练刚需，国产化率极低（长鑫存储在突破）。
- **CoWoS 先进封装**：AI 芯片产能瓶颈，台积电垄断，国产起步。

**Chiplet（芯粒）**：
- 先进制程受限下的"绕道"方案——用成熟制程 + 先进封装实现高性能。
- 连接 IC 设计与先进封装，是国产半导体的关键机会窗口。
- UCIe 联盟（英特尔/AMD/ARM/台积电）vs 中国 Chiplet 标准并行发展。

**第三代半导体（SiC/GaN）**：
- 新能源车（800V 高压平台 SiC）、快充（GaN）、射频（GaN）驱动。
- 国内外起步差距小（不像硅基芯片差距巨大），是"换道超车"机会。
- 壁垒在衬底生长（SiC 长晶极慢）和器件设计。

**RISC-V**：
- 开源指令集，x86/ARM 之外的第三极。
- 中科院"香山"/阿里"玄铁"等——国产 CPU 换道机会。
- 当前以 IoT / 嵌入式为主，高性能化（服务器级）仍在突破中。

### 维度 7：周期性判断与择时

> 半导体是典型的周期性行业（3-4 年一个周期），即使在国产替代主线下也逃不开周期。

**周期判断指标**：
- **全球半导体销售额**（月度，SIA 数据）——景气度风向标。
- **库存周转天数**——上升 = 去库存周期，下降 = 补库存周期。
- **产能利用率**——< 70% = 萧条，> 85% = 繁荣。
- **资本开支计划**——头部厂商（台积电/三星/中芯）capex 趋势。

**投资择时**：
- **周期底部**（产能利用率 < 70%、库存去化尾声）→ 最佳布局时机。
- **周期顶部**（产能满载、大量扩产）→ 警惕产能过剩。
- **国产替代标的**受周期影响相对小（结构性增长），但仍有β。

## 行业核心指标

| 指标 | 健康阈值 | 说明 |
|---|---|---|
| **国产化率** | 环节差异大（见维度 2） | 核心成长空间指标 |
| **头部客户认证状态** | 已进入头部晶圆厂产线 | **生死线** |
| **合同负债 / 预收款** | 持续增长 | 订单质量的领先指标 |
| **合同负债周转天数** | 缩短 = 订单加速 | **核心验证指标** |
| **毛利率** | > 35%（设备）/ > 25%（材料） | 技术壁垒体现 |
| **研发费用率** | > 15% | 技术驱动 |
| **良率 / 设备 OEE** | 达到或接近海外竞品 | "好用"的标志 |
| **产能利用率** | > 80% | 周期景气度 |
| **库存周转天数** | 持续下降 | 去库存/补库存信号 |
| **政府/大基金持股** | 有 | 资金与政策背书 |
| **资本开支 / 折旧** | 扩张期 > 1.5x | 成长投入 |
| **客户集中度** | 前 5 大客户 < 50% | 过高 = 议价权弱 |

## 壁垒判断

### 真壁垒
1. **已通过头部晶圆厂产线认证**——一旦进入，替换成本极高。
2. **从"能用"到"好用"的跨越**——良率/稳定性达到海外竞品水平。
3. **核心工艺 know-how**——半导体设备/材料的壁垒大量在"工程经验"而非纯理论。
4. **平台化产品线**——多品类设备/材料，单点突破 → 平台扩展。
5. **核心人才（海外大厂背景）**——应用材料/ASML/Lam Research 等资深工程师。
6. **大基金 / 国家队持股**——资金 + 政策 + 客户资源三重背书。

### 假壁垒（警惕）
1. **"国产替代"叙事但无头部客户认证**——政策概念 ≠ 商业落地。
2. **"对标 ASML/应用材料"但产品仍在研发**——样机阶段 ≠ 量产阶段。
3. **收入增长但合同负债不增**——可能是消化存量订单，非新订单增长。
4. **"卡脖子突破"但良率远低于海外**——卡在"能用"无法跨越到"好用"。
5. **政府补贴占利润 > 50%**——真实盈利能力存疑。
6. **估值靠"国产替代空间"测算但无客户进度支撑**——叙事型估值。

## 红旗识别
1. **宣称"突破 EUV / 7nm 以下先进制程"但无独立验证**——技术上极难，警惕夸大。
2. **头部晶圆厂认证进度停滞 > 2 年**——技术或客户关系有问题。
3. **合同负债下降但收入仍增长**——订单透支，增长不可持续。
4. **核心团队无半导体大厂背景**——半导体是经验密集型行业。
5. **"国产替代率"高但实际市场份额 < 5%**——概念性指标，非真实市占。
6. **政府补贴/退税占净利润 > 50%**——真实经营能力存疑。
7. **产品线单一且无扩展计划**——单一产品天花板低。
8. **客户集中度 > 70%（单一客户）**——议价权弱 + 替换风险。
9. **研发费用率 < 10% 却宣称"技术领先"**——投入与宣称不匹配。
10. **估值 PS > 20x 但收入 < 5 亿**——纯叙事估值。
11. **大基金未参与且无产业资本背书**——资金/资源孤立。
12. **管理层在 IPO 前大规模套现**——利益取向存疑。
13. **"对标英伟达/台积电"的初创设计公司**——巨头护城河极深，初创难类比。
14. **先进制程受限却宣称"不影响业务"**——低估制裁影响。
