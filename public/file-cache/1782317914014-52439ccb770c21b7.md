# 三表联动模型：利润表 / 资产负债表 / 现金流量表

这是 Anthropic 官方的 三表联动模型 skill。搭一套三大报表互相联动的财务模型：

- 利润表 → 留存收益 → 资产负债表

- 资产负债表 → 现金流量表

- 循环引用处理、配平检查（资产 = 负债 + 权益）

- 预测假设驱动：收入增长、毛利率、营运资金天数

适合：做公司财务预测、估值建模打底的人。三表联动是所有模型的基础。
