# 并购模型：测算收购的增厚 / 稀释效应

这是 Anthropic 官方的 并购（Merger）模型 skill。测算一笔收购对买方 EPS 的影响：

- 交易对价：现金 / 股票 / 混合，及对应的控制权

- 购买价格分摊（PPA）：商誉、无形资产识别

- 协同效应量化：收入协同 / 成本协同

- 增厚 / 稀释分析：未来几年 accretion / dilution

适合：做并购估值、评估一笔收购划不划算的人。上市公司并购的核心分析工具。
