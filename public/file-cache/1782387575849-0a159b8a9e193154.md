# 3 面 10 点行研框架（投资机构视角）

## 这是什么
一套面向 **PE/VC 投资决策** 的行业研究 SOP，把行研拆成 3 个层面（宏观/中观/微观）+ 10 个关键点，直接对应立项报告和 IC 备忘的章节结构。适合投资经理写行研报告、做赛道扫描、立项前做行业判断。

## 方法论框架

### 第 0 步：先明确行研目的（决定侧重点）
不同目的对应不同侧重，不要用一套模板打天下：

| 行研类型 | 侧重点 | 典型产出 |
|---|---|---|
| 发展趋势类 | 整体规模、细分规模、外部环境、驱动因素、风险 | 趋势预测报告 |
| 市场调研类 | 用户画像、需求痛点、竞品对比、渠道结构 | 市场进入策略 |
| 战略发展类 | 外部环境、产业链、壁垒与风险、关键成功因素 | 战略方向建议 |
| **投资策略类（PE/VC 主战场）** | 投资并购情况、估值水平、投资收益预测、退出路径 | 立项报告 / IC 备忘 |

### 第 1 层：宏观层面（外部环境）
**关键点 1 — 行业外部环境分析（PEST 模型）**
- **P 政治/政策**：监管和引导政策（扶持 vs 压制）、产业政策偏向、金融财政优惠（税收减免、财政补贴、低息贷款）、具体监管部门和监管要求。
- **E 经济**：GDP、人均可支配收入、宏观周期。
- **S 社会**：城镇化水平、人口结构、消费习惯演变（如单身经济、银发经济）。
- **T 技术**：科技发展情况、技术成熟度曲线。
- **中国特色重点**：政策及监管体系研究是判断行业发展趋势的基础。要判断国家对行业的态度和支持力度——这是中国市场不可跳过的一步。

### 第 2 层：中观层面（行业本身）
**关键点 2 — 行业概况**
- 行业定义（广义/窄义、细分边界）。
- 发展驱动因素：供给驱动（看产能，爆发力强）/ 需求推动（看潜在需求测算天花板）。
- 市场供需现状：供过于求 vs 供给不足。
- 发展阶段与周期：导入期/成长期/成熟期/衰退期。

**关键点 3 — 产业链与价值链分析**
- **产业链**：上中下游代表性企业、价值转移与增值机制、最终消费者是谁。
- **议价能力判断**：上游竞争激烈→下游议价能力强；下游竞争激烈→上游议价能力强。
- **价值链**：从原材料到成品每个环节的收入分成比例、成本结构拆解（深挖一家公司或一个产品）。

**关键点 4 — 市场规模测算**
- 从需求端或供给端测算；APP 类可测算用户规模辅助。
- 关键指标：市场容量、年增速、5 年 CAGR。
- **经验阈值**：规模千亿以上 + 增速 20%–30% + 远高于 GDP 增长 = 发展前景良好的行业。

**关键点 5 — 市场竞争格局**
- 行业集中度 CR4 / CR10，分级标准（见下表）。
- 竞争手段：价格战 vs 差异化。
- 波特五力模型分析。

| CR4 区间 | 市场类型 |
|---|---|
| > 75% | 极高寡占型 |
| 65%–75% | 高集中寡占型 |
| 50%–65% | 中（上）集中寡占型 |
| 35%–50% | 中（下）集中寡占型 |
| 30%–35% | 低集中寡占型 |
| < 30% | 原子型市场 |

**关键点 6 — 商业模式**
研发模式、采购模式、生产模式、销售模式、盈利模式。可用商业模式画布。

### 第 3 层：微观层面（公司）
**关键点 7 — 龙头公司分析（标杆对比）**
- 上市公司：财务信息最重要（收入结构、毛利、净利、增速、成本、现金流、存货、研发投入）。
- 非上市公司：成立时间、团队、发展历程、主要产品、商业模式、核心经营数据、融资、资方、优势、挑战、未来战略。

**关键点 8 — 目标公司深度分析**
比龙头公司更深一层，必须覆盖：基本情况、业务情况、行业情况、财务情况（三张表 + 前五大供应商/客户 + 关联交易）、法务情况（未决诉讼/法律纠纷）、治理结构、发展战略、股权结构（国资比例/混改/实控人）、团队情况、资本运作（投融资历史、估值、上市/并购）。

**关键点 9 — 发展趋势**
- 行业趋势：未来市场规模、格局、技术更迭、供需、商业模式、用户群体。
- 公司业绩预测：中外对比（经济时差）、相似行业对比。

**关键点 10 — 机会与风险**
- **机会**：基于趋势预测和行业痛点，找出细分投资机会。
- **风险**：必须点名（如版权风险、安全风险、核心供应商依赖风险）。

## 数据源推荐（清单）
**一手信息**：访谈、问卷、实地走访。
**二手信息**：
- 一级市场：鲸准、烯牛数据、企名片、清科私募通、IT 桔子。
- 二级市场：Wind、东方财富网、慧博。
- 研报：启明星、严究、萝卜投研。
- 上市公司公告、统计局、一行三会数据。

## 占位符
- `[行研目的]`：投资决策 / 战略规划 / 市场进入 / 趋势预测之一。
- `[行业定义]`：广义 + 窄义 + 细分边界。
- `[CR4 数值]`：填入后对照分级表。
- `[市场规模测算公式]`：需求端法 或 供给端法。
- `[龙头公司清单]`：列出 3–5 家对标公司。
- `[目标公司核心问题清单]`：针对尽调目标列出 10–15 个必答问题。

## 红旗识别（这些信号说明行业/项目可能有问题）
1. **CR4 长期低于 30% 且无集中趋势**：原子型市场，除非有强规模效应，否则难出大公司（如生猪、餐饮）。
2. **政策口径不明或近期有收紧信号**：监管方向是最大不确定性，常见于教育、医美、金融科技、平台经济。
3. **市场规模增速低于 GDP**：说明行业已进入成熟甚至衰退期，难有超额回报。
4. **龙头公司市场份额持续下滑**：说明行业格局在重构，新进入者正在颠覆。
5. **产业链利润严重向上游或下游倾斜**：中游环节可能是"苦力"，毛利被挤压。
6. **目标公司前五大客户/供应商集中度过高（> 50%）**：客户/供应商依赖风险。
7. **关联交易频繁且无合理解释**：利润转移、利益输送嫌疑。
8. **融资金额远大于实际业务需要**：可能是估值游戏，而非真实业务扩张。
9. **核心团队近期频繁变动**：内部治理问题。
10. **"供给驱动"行业但产能利用率低于 60%**：说明需求被高估。
