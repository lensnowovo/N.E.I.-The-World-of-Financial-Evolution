# IC 投委会 memo：把尽调结论写成投资建议书

这是 Anthropic 官方的 PE 投委会 memo skill。把尽调发现、财务分析、交易条款综合成一份结构化的 IC 投资建议书：

- 投资逻辑 / 商业模式 / 市场机会

- 财务亮点与风险

- 估值与交易结构

- 投资建议与条件

适合：要把一个项目推向 IC 决策的投资人。让 Claude 出 memo 初稿，你做判断和润色。
