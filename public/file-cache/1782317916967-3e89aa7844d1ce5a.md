# Excel 模型审计：查硬编码、断链、不平的公式

这是 Anthropic 官方的 Excel 模型审计 skill。拿到一份别人的模型，快速查出问题：

- 硬编码检测：哪些数字是手敲的、没链接到假设

- 公式追踪：断链、循环引用、错误值（#REF / #DIV）

- 配平检查：三表是否真的平衡

- 一致性检查：同一指标在不同 sheet 用不同算法

适合：接手别人模型、或自己模型出 bug 找不到原因的人。尽调时核查管理层模型也用得上。
