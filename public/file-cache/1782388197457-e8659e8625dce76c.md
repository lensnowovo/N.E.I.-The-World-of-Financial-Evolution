# Crypto / Web3 投资研究框架（DeFi / Tokenomics / 链上数据）

## 这是什么
加密货币 / Web3（DeFi / 代币经济 / 链上数据 / RWA）的投资分析框架。适合 **Crypto Fund / 海外配置型 VC / FoFs** 做协议级尽调。核心反直觉：**Web3 最常用的指标 TVL 是一把"有设计缺陷的尺子"——同一笔钱可能被数 3 遍**。专业的 Crypto 投资必须穿透 TVL 水分，看协议真实的**费用收入、用户活跃度、资本效率**三件套。

## 行业研究框架（6 维度）

### 维度 1：协议的商业变现能力（从 TVL 到"谁在赚钱"）

**反共识结论**：TVL 大 ≠ 协议赚钱。

| 案例 | TVL | 年化费用收入 | 启示 |
|---|---|---|---|
| Uniswap | 较低 | 1.11 亿美元 | 费用开关打开后变现能力强 |
| Balancer | 高 33% | 3080 万美元 | TVL 高但变现弱 |

**三层收入定义**（必须区分）：
1. **协议费用（Fees）**：用户使用协议支付的全部费用。
2. **协议收入（Revenue）**：扣除给流动性提供者后，协议国库/代币持有者实得的部分。
3. **持币者收入（Holders Revenue）**：通过回购销毁/质押分红实际分配给代币持有者的部分（类似股息 + 回购）。

**关键判断**：很多协议 Fees 很大，但因几乎全部分配给 LP，Revenue 少得可怜——这类协议的代币价值捕获能力存疑。

### 维度 2：TVL 水分拆解（专业必修课）

**TVL 三大注水来源**：

1. **收益聚合器重复计算**：用户存入 Convex → 资金流向 Curve → 同一笔钱被 Convex 和 Curve 各计一次 TVL。极端案例：Solana 上 Saber + Sunny 聚合器占 75 亿 TVL（全链才 105 亿）。

2. **流动性质押衍生品（LSD）分身术**：1 枚 ETH → stETH（Lido）→ 存入 Aave 当抵押 → Curve 流动性池 → EigenLayer 再质押。**同一笔 ETH 同时出现在 4 份报表里**。Lido 的 stETH 中：21.6% 存 Aave、14.7% 在 Curve 池。

3. **价格波动幻影**：TVL 是价格的函数。ETH 从 3400 → 2400，全链 TVL 从 1200 亿 → 860 亿——没有资金流出，只是估值缩水。

**学术实证**：UCL + 南洋理工对 3570 个协议的研究——**2021 年 12 月峰值时，TVL vs TVR（真实可赎回价值）差额 1398 亿美元，虚增比例约 50%**。

**替代指标**：
- **TVR（Total Value Redeemable）**：只计纯代币，排除衍生品 + 借入代币。
- **独立存款地址数**：50 个巨鲸撑起的 TVL vs 20 万散户的 TVL——生态健康度完全不同。
- **TVL / FDV 比值**：比值过低 = 协议估值远超实际使用规模。

### 维度 3：代币经济学（Tokenomics）

**供给端分析**：
- **市值（Market Cap）= 当前价格 × 流通供应**。
- **完全摊薄估值（FDV）= 当前价格 × 最大供应**。
- **市值 / FDV 比值**：比值 10% + 一年内全解锁 = 需要增长 10 倍才能维持现价（巨大抛压）；比值 25% + 4 年解锁 = 年化 40% 增长即可。

**代币类型**：
- **权益代币（Security Token）**：代表资产所有权，受证券法约束。
- **实用型代币（Utility Token）**：平台内功能使用，不直接代表资产。
- **混合型**：两者结合。

**代币模型 4 大要素**：
1. **价值累积机制**：回购销毁 / 质押分红 / Fee Switch 开关。
2. **通胀/通缩**：铸造速率 + 销毁速率的动态平衡。
3. **解锁节奏**：团队/投资人/生态激励的解锁时间表。
4. **防 Dump 机制**：锁仓、惩罚性税率、手续费回流。

**两种典型设计**：
- **社区驱动型**：总量大（100T）+ 高空投（50%+）+ 项目方低占比（< 10%）—— 适合 meme，但需要高频运营。
- **增发控制型**：初始发行小（10M）+ 多签/DAO 控制增发 + 放弃铸币权——提升市场信任。

### 维度 4：用户真实度与链上行为

**关键指标**：
- **独立存款地址数**：剔除巨鲸后的真实用户规模。
- **活跃地址数 / 持币地址数**：用户粘性。
- **交易频次与金额分布**：是否少量地址刷量。
- **新地址增长率**：真实用户获取。

**链上数据工具栈**：
- **DefiLlama**：TVL/费用/收入跨协议对比（已做去重调整）。
- **Dune Analytics**：自定义 SQL 查询链上数据。
- **Token Terminal**：协议财务数据（类似 Web3 版 Bloomberg）。
- **Nansen**：地址标签 + 巨鲸追踪。
- **Glassnode / CryptoQuant**：链上指标 + 交易所流量。

### 维度 5：国库与现金流

**国库（Treasury）分析**：
- 国库规模 + 资产构成（稳定币 vs 协议代币 vs 蓝筹）。
- 月度燃烧率（burn rate）——能撑多久。
- 是否通过代币发行维持运营（不可持续）。

**现金流视角**：
- 协议真实收入能否覆盖运营成本？
- 代币增发补贴用户的比例——补贴结束后的留存率。

### 维度 6：监管与合规

- **美国**：Howey Test 四要素判定证券属性；CFTC vs SEC 管辖权。
- **欧盟**：MiCA 法案 2024 年全面实施——全球首个完整加密监管框架。
- **香港/新加坡/迪拜**：牌照化合规路径（VASP 牌照）。
- **中国**：境内禁止加密交易 + 香港 Web3 试点——"一国两制"格局。
- **稳定币监管**：美国 GENIUS Act、欧盟 MiCA——稳定币发行牌照化。

## 行业核心指标

| 指标 | 健康阈值 | 说明 |
|---|---|---|
| **协议年化费用收入** | > 1000 万美元 | 真实商业规模 |
| **协议收入 / FDV** | > 2% | 类似"Earnings Yield" |
| **TVL / FDV** | > 5% | 使用规模 vs 估值 |
| **市值 / FDV** | > 40% | 解锁抛压风险（越低越危险） |
| **独立存款地址数** | > 1 万 | 剔除巨鲸的真实用户 |
| **月活跃地址 / 持币地址** | > 30% | 用户活跃度 |
| **资本利用率**（借贷协议） | > 30% | 资金效率 |
| **国库 runway** | > 24 个月 | 不靠代币发行能撑多久 |
| **质押率** | 20-60% | 太低无共识，太高流动性差 |

## 壁垒判断

### 真壁垒
1. **网络效应**：DEX/借贷协议的流动性网络——资金越深，滑点越低，用户越多。
2. **安全审计与历史记录**：无重大黑客事件的协议获得信任溢价（Lido/Aave）。
3. **代币价值累积机制**：真实回购销毁/分红的协议（vs 纯治理代币）。
4. **开发者生态**：GitHub 活跃度、协议扩展数、第三方集成数。
5. **合规先行优势**：在 MiCA/GENIUS Act 下率先获牌的稳定币/交易所。

### 假壁垒（警惕）
1. **高 TVL 数字**：可能是聚合器/LSD 重复计算的幻影。
2. **空投用户的"高活跃"**：空投撸毛党，空投结束即流失。
3. **"代币激励换 TVL"**：停止激励后 TVL 雪崩——不可持续。
4. **名人/大 V 背书**：叙事驱动而非基本面。

## 红旗识别
1. **TVL 增长主要靠代币激励**——激励停止即崩盘（如早期 Curve Wars）。
2. **市值 / FDV < 10%** + **大量团队/投资人代币即将解锁**——巨大抛压。
3. **独立存款地址数 < 1000 但 TVL > 1 亿**——少数巨鲸操纵。
4. **代币无价值累积机制**（纯治理）——价值捕获为零。
5. **国库 80%+ 是自家代币**——熊市时国库贬值，运营瘫痪。
6. **核心开发者近期离职 + GitHub 提交骤降**——项目死亡信号。
7. **协议合约未经多次审计 + 匿名团队**——黑客/跑路风险。
8. **"RWA/DePIN/AI"多概念叠加但无实际收入**——纯叙事。
9. **创始人频繁在社交媒体喊单**——操纵市场嫌疑。
10. **监管明确禁止的业务模式**（如未注册证券发行）——法律黑天鹅。
