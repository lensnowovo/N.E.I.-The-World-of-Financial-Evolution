# 2026 头部 VC 投前尽调 8 大新标准（硬科技/国资基金时代）

## 这是什么
反映红杉、高瓴、深创投、IDG 等一线 VC 在 2026 年实际使用的 **内部立项打分体系**。传统"团队/赛道/财务/法律"四项仅占 40% 权重（及格门槛），新增 8 大硬核指标占 60% 权重（决定过会、估值、能否拿到钱）。适合创业者、投资经理、FA 了解 **当前 VC 真实底层打分逻辑**，避免按旧逻辑写 BP 跑路演。

## 背景：为什么旧逻辑失效
- **旧逻辑（适配轻资产、模式创新、互联网流量）**：看人、讲故事、赌增量。豪华团队 + 万亿赛道 + 干净财法 = 轻松拿钱。
- **新逻辑（适配硬科技、重资产、强政策）**：美元基金收缩，人民币国资基金主导；投资主线转向硬科技/高端制造/算力基建/生物医药——长周期、重研发、强政策。
- **传统四项**：团队背景、赛道 TAM、财务增长、法律合规——只是不被淘汰的入门门槛。
- **新增八大**：算力数据合规、供应链安全、ESG、国产替代率、专利壁垒、客户粘性、退出确定性、政策风险敞口——决定评级、估值、能否过会。

## 方法论框架（8 大新增指标）

### 指标 1：算力 & 数据合规（AI 项目第一准入红线）
**核查要点**：
- 数据采集、存储、传输、跨境全流程合规。
- 训练数据是否存在版权侵权。
- 算力部署是否契合国家调度规则。
- 隐私保护、合规备案是否齐全。

**红旗信号**：AI 项目技术过硬、场景成熟，但数据来源模糊、合规备案缺失 → 直接止步首轮立项。

### 指标 2：供应链安全（硬科技企业核心生命线）
**核查要点**（穿透全产业链）：
- 核心部件是否过度依赖海外厂商。
- 有无断供、卡脖子潜在风险。
- 是否完成国产备份供应链布局。

**国资机构特别要求**：直接将"供应链自主可控"纳入硬性打分项，优先重仓双供应链布局 + 国产替代进度明确的企业。

### 指标 3：ESG 硬指标（从品牌门面变成融资必修课）
**核查要点**：环保排放达标、安全生产规范、用工合规体系、碳中和布局、社会责任建设。
**重点关注行业**：制造、化工、能源——环保不达标/安全隐患/用工纠纷 = 一票否决。

### 指标 4：国产替代率（决定政策红利与估值天花板）
**核查方式**（精准测算，非笼统看赛道）：
- 对标海外竞品的性能、定价、市场份额。
- 评估替代空间、落地节奏、导入头部大厂供应链进度。

**估值溢价条件**：替代空间广阔 + 已批量落地 + 领跑行业替代节奏。
**淘汰条件**：单纯复刻海外技术 + 无核心差异化 + 替代价值薄弱。

### 指标 5：专利壁垒强度（从拼数量转向拼硬核质量）
**核查要点**：
- 核心专利是否为底层原创。
- 有无侵权隐患。
- 全球专利布局完整性。
- 能否构建行业护城河。
- 剔除凑数专利、垃圾专利、与主业无关的专利。

### 指标 6：客户粘性（复购续约率 > 短期增速）
**核查要点**（紧盯，而非只看营收）：
- 标杆客户复购率。
- 年度续约率。
- 客户留存结构。
- 单客户 LTV 终身价值。

**优质画像**：手握国企/产业龙头长期稳定订单 + 复购续约健康。
**脆弱画像**：依赖零散小单 + 一次性合作 + 无持续付费能力。

### 指标 7：退出确定性（提前锁定 3–5 年资本路径）
**投前推演**：
- 是否符合科创板/创业板/港股 18C 上市标准。
- 有无产业 CVC 并购意向。
- 是否具备老股转让、S 基金接盘基础。

**优质项目**：同时满足上市标准 + 产业并购价值 → 退出确定性高，更易获重仓。

### 指标 8：政策风险敞口（规避行业黑天鹅）
**重点核查**：业务是否触碰监管红线、行业政策是否在收紧、商业模式是否契合国家产业导向。
**高风险赛道（需谨慎观望）**：生成式 AI、医美、教培、金融科技、平台经济。
**政策红利赛道**：高端制造、双碳目标、卡脖子技术突破方向。

## 打分权重总览

| 维度 | 权重 | 作用 |
|---|---|---|
| 传统基础项（团队/赛道/财务/法律） | 40% | 基础筛除，判定"能不能活下去" |
| 新增 8 大指标 | 60% | 决定评级/估值/能否过会，判定"能不能做大、合规经营、穿越周期、安全退出" |

## 占位符
- `[项目/公司名称]`
- `[赛道]`：AI / 硬科技 / 生物医药 / 新能源 / 高端制造等。
- `[资方类型]`：美元基金 / 人民币国资基金 / 市场化人民币基金 / CVC。
- `[国产替代率]`：对标海外产品的替代百分比。
- `[核心专利清单]`：底层原创专利数量 + 全球布局情况。
- `[前 5 大客户复购率]`：续约率数据。
- `[退出路径推演]`：科创板 / 创业板 / 港股 18C / 并购 / S 基金。
- `[政策风险等级]`：低 / 中 / 高。

## 红旗识别（一票否决项）
1. **AI 项目数据来源模糊或合规备案缺失**——技术再好也过不了。
2. **核心部件单一海外供应商，无国产备份**——卡脖子风险。
3. **制造/化工/能源项目环保不达标或有重大安全隐患**——国资基金直接一票否决。
4. **专利数量多但均为凑数/外围/无关专利**——虚假壁垒。
5. **营收主要靠补贴、一次性订单、无复购**——商业模式脆弱。
6. **所处赛道政策在收紧（教培/医美/平台经济类）**——黑天鹅风险。
7. **无清晰上市预期且无产业并购价值**——退出路径堵塞。
8. **过度依赖海外供应链却未启动国产替代布局**——在国资主导市场将被边缘化。
9. **创始团队只讲传统四项（团队/赛道/财务/法律），对 8 大新指标避而不谈**——融资认知落后于市场。
