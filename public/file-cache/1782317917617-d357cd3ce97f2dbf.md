# CIM 招股书撰写：把项目卖点写成买家愿看的备忘录

这是 Anthropic 官方的 CIM（Confidential Information Memorandum） skill。FA 卖项目时写的那本招股书：

- 执行摘要 · 投资亮点 · 公司概况

- 行业机会 · 商业模式 · 财务摘要

- 增长策略 · 交易结构

- 根据买家类型（战略 / 财务）调整叙事重点

适合：做 FA 卖方、或融资方整理给投资人的材料。CIM 是交易执行的核心交付物。
