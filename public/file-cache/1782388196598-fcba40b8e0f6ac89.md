# 金融科技投资研究框架

## 这是什么
金融科技（支付/信贷/保险科技/数字银行/监管科技/稳定币）的投资分析框架。适合 **PE/VC/产业资本** 做金融科技赛道扫描与全球机会判断。金融科技是 **诞生独角兽最多的行业之一**（全球 326 家，占独角兽总数 > 1/4），核心矛盾是：**监管套利时代结束，企业必须在"持牌做金融"与"纯科技赋能"两条路之间二选一**。

## 行业研究框架（6 维度）

### 维度 1：监管套利空间与牌照价值
**核心判断**：全球主要国家已完成金融科技监管框架构建，纯监管套利的空间大幅收窄。

**两条转型路径**（企业必须二选一）：

| 路径 | 特征 | 案例 | 估值逻辑 |
|---|---|---|---|
| **A. 持牌做金融** | 申请/收购银行/支付/保险牌照，转型为金融机构 | Zopa（英国银行牌照）、SoFi（收购 Golden Pacific Bank）、Lending Club（收购 Radius Bank） | 金融机构估值（PE 10-15x） |
| **B. 纯科技赋能** | 完全 2B，向银行/保险机构输出技术服务 | Plaid、Marqeta、Adyen | 科技公司估值（PS 5-10x 或 PE 20-30x） |

**中国特殊语境**：P2P 已 2020 年清零，"非持牌做金融"路径基本关闭——持牌 or 科技，无中间地带。

### 维度 2：细分赛道投资逻辑

| 细分 | 全球 VC 投资活跃度 | 核心逻辑 | 关注点 |
|---|---|---|---|
| **银行/借贷科技** | ★★★★★（116 亿美元） | 新兴市场银行服务不足 | 获客成本、不良率、资金成本 |
| **证券/资管科技** | ★★★★（88 亿美元） | 财富管理民主化 | AUM、用户粘性、ARPU |
| **支付科技** | ★★★★ | 跨境/实时/低成本 | GMV、take rate、合规 |
| **保险科技** | ★★★ | 风控 + 场景化产品 | 赔付率、续保率、渠道 |
| **加密/区块链** | ★★★★★（按交易数） | 基础设施投资 | 见 Crypto 框架 |
| **监管科技** | ★★★ | 合规自动化 | 监管变化驱动 |

### 维度 3：新兴市场 vs 成熟市场
**新兴市场（东南亚/拉美/中东/非洲）是金融科技独角兽主战场**：
- **拉美**：金融科技占全部 VC 投资的 57%；巴西只有 10.5% 成年人有信用卡；Nubank 市值数百亿美元。
- **东南亚**：新加坡/印尼独角兽最多（Akulaku、MoMo、Mynt）；传统金融服务覆盖缺口大。
- **中东/非洲**：20 亿人口、65% < 30 岁；52% 无银行账户；稳定币已普及（尼日利亚 200 亿美元跨境汇款）。
- **成熟市场**：美国独角兽 171 家（> 全球一半）；2025 年迎上市潮（Chime/Circle/eToro/Klarna）。

### 维度 4：单位经济学（核心）
| 业务类型 | 关键 KPI | 健康阈值 |
|---|---|---|
| 数字银行 | CAC、LTV、活期用户 ARPU | LTV/CAC > 3 |
| 信贷科技 | 获客成本、不良率（NPL）、资金成本 | NPL < 5%、资金成本 < 8% |
| 支付 | GMV、take rate、拒付率 | take rate 1-3%、拒付 < 1% |
| 保险科技 | 赔付率、综合成本率、续保率 | 综合成本率 < 100% |
| 财富管理 | AUM、管理费率、用户粘性 | 管理费 0.25-1% |
| BNPL | GMV、不良率、商家承担比例 | 不良率 < 3% |

### 维度 5：AI 与新技术驱动
- **生成式 AI**：深度渗透信贷风控、智能投顾、反欺诈、客户服务。
- **稳定币**：跨境支付成本效率优势已超过 Visa/Mastercard。
- **开放银行**：PSD2 等法规推动 API 化银行服务。
- **Web3/DeFi**：去中心化金融基础设施与传统金融融合（RWA 代币化）。

### 维度 6：大型金融机构布局逻辑（退出端）
- **摩根大通**：run the bank + change the bank 双轨；量子计算 + JPM Coin。
- **高盛**：Marcus、Marquee、TxB、Financial Cloud；谨慎拥抱 AI。
- **贝莱德**：Aladdin 平台为核心；投资 Oportun/LendingClub/Funding Circle。
- **花旗**：Citi Ventures 2024 年投资数全球第一；偏好证券资管、银行借贷、支付科技。

**退出启示**：被大型金融机构并购/投资是金融科技公司的重要退出路径——早期即应研究哪家机构最可能接盘。

## 行业核心指标

| 指标 | 应用场景 | 健康阈值 | 说明 |
|---|---|---|---|
| **CAC（获客成本）** | 所有 2C 业务 | < ARPU × 3 | 持续走高 = 流量红利消退 |
| **LTV/CAC** | 所有 2C 业务 | > 3 | < 1.5 = 商业模式不成立 |
| **不良率（NPL）** | 信贷 | < 5% | > 8% = 风控失灵 |
| **资金成本** | 信贷/数字银行 | < 8% | 持牌机构优势 |
| **综合成本率** | 保险 | < 100% | > 100% = 承保亏损 |
| **GMV + take rate** | 支付 | take rate 1-3% | |
| **AUM 增速** | 财富管理 | > 20% | 净流入 vs 市场增值拆分 |
| **月活/日活** | C 端 App | DAU/MAU > 30% | |
| **复购率/续保率** | 订阅/保险 | > 70% | |
| **监管合规成本占比** | 全行业 | < 营收 10% | 过高 = 业务模式被监管侵蚀 |

## 壁垒判断

### 真壁垒
1. **金融牌照**：银行/支付/保险/证券牌照——稀缺、申请周期长（1-3 年）。
2. **数据飞轮**：用户交易/信贷数据越多，风控模型越准——先发优势。
3. **网络效应**：支付网络（商户 × 用户双边）、社交投资平台。
4. **合规能力**：反洗钱/KYC/反欺诈基础设施——监管越严越值钱。
5. **资金成本优势**：持牌银行存款资金 vs 非持牌机构批发资金——成本差 3-5 个百分点。

### 假壁垒
1. **"流量红利"**：靠补贴获客，无粘性——补贴停就流失。
2. **"技术领先"**：纯技术无牌照/数据护城河——可被复制。
3. **"大 V/网红渠道"**：流量成本终将趋同于投放渠道 ROI。

## 红旗识别
1. **无牌照做金融业务**——监管风险一票否决（中国语境下）。
2. **不良率持续攀升且管理层粉饰**——风控模型失效。
3. **CAC > LTV/3**——单位经济不成立。
4. **资金成本 > 12%**——资金端无优势，信贷业务难盈利。
5. **业务模式依赖单一监管漏洞**——漏洞修补即业务崩塌。
6. **管理层频繁变动 + 合规事故**——治理问题。
7. **应收账款/在途资金巨大且不透明**——资金挪用风险。
8. **海外业务收入占比高但当地监管环境不明**——政策黑天鹅。
9. **宣称 AI 驱动但风控人员 < 20 人**——AI 是包装。
10. **关联金融机构利益输送**——牌照寻租而非真实科技价值。
