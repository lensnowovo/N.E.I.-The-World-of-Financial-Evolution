# 招投标与订单信息检索方法论 · Skill 草稿

## 这是什么

面向 PE/VC/FA 投研人员的**中国招投标信息检索与订单真实性验证 SOP**。To B / To G（政府/国企）商业模式的公司，招投标公告是最真实的需求与收入验证渠道——远比公司自己宣称的"订单"可信。一份中标公告能告诉你：谁在采购、采购什么、采购多少、中标价、中标方、竞争格局。但很多投研人查招投标只会搜百度，不知道全国公共资源交易平台、不会用寻标宝/标事通、不懂政府/国企/军队的不同采购体系、不会从招投标数据反推公司收入真实性。本 Skill 把招投标检索拆成"选平台 → 检索 → 验证收入 → 识别风险"四步，可作为任何 To B / To G 公司尽调的标准动作。适合投 To B / To G（政府/国企/军队采购）/ 医疗器械 / 军工 / 信创 / 环保 / 教育的公司、FA、政府事务。

## 方法论框架（6 维度）

### 维度 1：招投标信息源全景图（去哪查）

> 中国招投标信息分散在多个体系——按采购方类型选择对应平台。

#### 按采购方类型的平台选型矩阵

| 采购方类型 | 首选平台 | 特点 | 信息完整度 |
|---|---|---|---|
| **政府机关** | 中国政府采购网 ccgp.gov.cn + 各省政府采购网 | 法定公开，最规范 | ★★★★★ |
| **国企/央企** | 中国招标投标公共服务平台 ctbpsp.com + 企业自有招标平台 | 依法必招项目全公开 | ★★★★ |
| **军队/军工** | 军队采购网 plap.mil.cn + 全军武器装备采购信息网 weain.mil.cn | 涉密项目少公开，但有框架协议 | ★★ |
| **事业单位** | 各类公共资源交易中心 | 部分公开 | ★★★ |
| **大型民企** | 企业自有招标平台（如华为/阿里/腾讯供应商门户） | 非强制公开 | ★★ |

#### 聚合/商业检索平台（一站式）

| 平台 | 特点 | 收费 |
|---|---|---|
| **全国公共资源交易平台 ggzy.gov.cn** | 国家级，覆盖最全 | 免费 |
| **中国招标投标公共服务平台 ctbpsp.com** | 依法必招项目全收录 | 免费 |
| **中国政府采购网 ccgp.gov.cn** | 政府采购权威 | 免费 |
| **寻标宝（百度旗下）xunbiaobao.baidu.com** | 聚合 + 智能检索 | 部分免费 |
| **标事通 biaoshitong.ccpc360.com** | 聚合 + 订阅 | 付费 |
| **标找找（企查查旗下）biaozhaozhao.com** | 聚合 + 与企业信息打通 | 部分免费 |
| **千里马招标网 qianlima.com** | 老牌聚合 | 付费 |
| **剑鱼标讯 jianyu360.com** | 聚合 + 移动端 | 部分免费 |

**投研建议**：
- **免费基础**：全国公共资源交易平台 + 中国政府采购网 + 中国招标投标公共服务平台。
- **深度分析**：寻标宝/标事通/标找找（支持订阅 + 智能分析）。
- **企业关联**：标找找（与企查查数据打通，可关联企业股权/风险信息）。

### 维度 2：招投标全流程与公告类型（查什么）

> 招投标是一个多阶段流程，每个阶段都有不同公告——理解流程才能找对信息。

#### 政府采购/招投标公告类型矩阵

| 公告类型 | 阶段 | 关键信息 | 投研价值 |
|---|---|---|---|
| **采购意向公告** | 最早期 | 采购计划、预算金额、预计时间 | ★★★★★（提前 3-6 月知道需求） |
| **招标公告** | 正式启动 | 采购需求、资格要求、投标截止 | ★★★★ |
| **资格预审公告** | 入门筛选 | 投标方资质门槛 | ★★★ |
| **中标候选人公示** | 评标后 | 第一/二/三中标候选人 + 报价 | ★★★★★（竞争格局） |
| **中标公告** | 最终 | 中标方、中标金额、标的明细 | ★★★★★（最权威） |
| **合同公告** | 签约后 | 合同金额、履约期、付款方式 | ★★★★★（真实收入） |
| **废标公告** | 异常 | 流标原因（资格不符/竞争不足等） | ★★★（行业信号） |

**投研核心关注**：
- **中标公告 + 合同公告**：最权威的真实订单数据。
- **采购意向公告**：提前 3-6 月预判需求（最早信号）。
- **中标候选人公示**：看竞争格局（几家中标、价格差距）。

### 维度 3：检索技巧（怎么搜）

#### 关键词策略

| 检索目的 | 关键词组合 |
|---|---|
| **找某公司所有中标记录** | 公司名称（精确匹配） + 限定"中标公告" |
| **找某产品/服务的采购** | 产品名/服务名 + 限定地域/时间 |
| **找某行业的采购趋势** | 行业关键词 + 时间范围 + 按金额排序 |
| **找某客户的采购偏好** | 客户名 + 全时段 + 分析中标方分布 |

#### 平台检索示例

```
# 中国政府采购网
"XX公司" 中标公告 2023-2025

# 寻标宝/标事通（支持高级筛选）
公司名 = "XX科技"
公告类型 = 中标公告
时间 = 近 3 年
排序 = 金额降序

# 全国公共资源交易平台
关键词 = "医疗器械"
地域 = "广东省"
时间 = 近 1 年
```

#### 智能订阅

寻标宝/标事通/标找找支持：
- **关键词订阅**：产品/客户/竞品名——新公告邮件/APP 推送。
- **企业监控**：监控某公司新中标——竞品追踪。
- **地域订阅**：监控某省/市采购动态——地方政府客户追踪。

### 维度 4：用招投标数据验证公司收入（核心应用）

> 这是投研最高价值的应用——To B/To G 公司宣称的收入能否经得起招投标数据交叉验证？

#### 收入验证 4 步法

##### 步骤 1：拆解公司宣称的客户结构

从公司 BP/年报中提取：前 5/10 大客户名称 + 宣称收入占比 + 宣称合同金额。

##### 步骤 2：在招投标平台反查每个客户

- 搜索客户名 + 目标公司名 → 找到对应中标公告/合同公告。
- 比对：**公告金额 vs 宣称金额** → 是否匹配？

##### 步骤 3：找"漏网"订单

- 搜客户名（不限中标方）→ 看该客户同期还采购了什么、给了哪些供应商。
- 如果目标公司宣称 1 亿订单但招投标系统只有 3000 万 → **70% 收入存疑**。
- 如果客户根本没在招投标平台出现 → 可能是框架协议/直接采购（需进一步问公司）。

##### 步骤 4：分析竞争格局

- 搜客户名（全时段）→ 看历次中标方分布。
- 判断目标公司是真"核心供应商"还是偶尔中标。
- 判断是否有多家关联公司"陪标"（疑似造假）。

#### 验证结论分类

| 结论 | 数据特征 | 投研判断 |
|---|---|---|
| **✅ 高度吻合** | 招投标金额 ≈ 宣称收入（±20%） | 收入真实 |
| **⚠️ 部分吻合** | 招投标覆盖 50-80% 宣称收入 | 剩余部分需追问（框架/直接采购？） |
| **🔴 严重不符** | 招投标覆盖 < 30% 宣称收入 | 收入真实性存疑，需深度调查 |
| **🚫 完全查不到** | 客户/订单在招投标平台无任何痕迹 | 极高风险——可能虚构客户 |

### 维度 5：从招投标数据看行业竞争格局（高阶应用）

#### 竞争格局分析 4 维度

1. **中标方集中度**：前 3/5/10 名中标方占比 → 是垄断还是分散？
2. **中标价格趋势**：同类采购历年中标价变化 → 价格战？技术降本？
3. **新进入者频率**：每年新出现的中标方 → 行业门槛高低。
4. **地域分布**：中标方地域集中度 → 地方保护 vs 全国竞争。

#### 实战场景

- **信创/国产替代进度**：搜"国产 CPU/操作系统/数据库"在政府/国企采购的中标记录 → 真实替代率。
- **医疗器械集采影响**：搜集采前后同类产品中标价变化 → 价格压力量化。
- **军工配套商识别**：搜军队采购网中标方 → 真实进入军工供应链的公司。

### 维度 6：识别风险信号（红旗应用）

#### 招投标数据揭示的风险信号

| 风险信号 | 数据特征 | 含义 |
|---|---|---|
| **虚假订单** | 公司宣称大订单但招投标系统无记录 | 可能虚增收入 |
| **陪标造假** | 多家关联公司同时投标，价格梯度整齐 | 涉嫌围标 |
| **单一客户依赖** | 同一客户占公司中标总额 > 70% | 客户集中风险 |
| **订单骤降** | 近 1 年中标数量/金额断崖式下降 | 业绩恶化前兆 |
| **价格战** | 同类采购中标价持续下降 | 行业利润承压 |
| **频繁废标** | 同类采购多次废标 | 行业供需/资质问题 |
| **新进入者抢标** | 不知名新公司低价中标 | 行业门槛下降 |
| **地域集中** | 中标集中于单一省份 | 地方保护主义/关系依赖 |

## 行业核心指标

| 指标 | 健康阈值 | 说明 |
|---|---|---|
| **招投标覆盖收入比** | > 70% | 收入可验证性 |
| **核心客户中标记录** | 可查、金额匹配 | 客户真实性 |
| **中标集中度（前 5）** | < 60% | 避免单一客户依赖 |
| **近 3 年中标数量趋势** | 稳定或增长 | 业务可持续性 |
| **中标价格趋势** | 稳定或合理下降 | 非价格战 |
| **竞争格局** | 多家良性竞争 | 无垄断/无陪标 |
| **地域分布** | 跨省覆盖 | 非地方关系依赖 |
| **采购意向前瞻** | 有未来 3-6 月需求 | 需求可见性 |

## 红旗识别
1. **宣称收入 vs 招投标数据严重不符**（覆盖 < 30%）——收入造假最大风险。
2. **客户在招投标平台完全查不到**——客户可能虚构。
3. **单一客户占中标总额 > 70%**——客户集中风险。
4. **多家关联公司同时投标**——涉嫌围标/陪标造假。
5. **中标数量近 1 年断崖下降**——业绩恶化前兆。
6. **中标价持续下降 > 30%/年**——行业价格战，利润承压。
7. **同类采购频繁废标**——行业供需/资质问题。
8. **不知名新公司低价抢标**——行业门槛下降，壁垒消解。
9. **中标地域高度集中于单一省份**——地方关系依赖，非真实竞争力。
10. **宣称"政府大客户"但政府采购网无记录**——客户宣称造假。
11. **宣称"军工订单"但军队采购网无记录**——涉密豁免，需其他验证。
12. **合同金额 vs 中标金额严重偏离**——后续变更/取消。

## 占位符
- `[目标公司]`：被尽调的 To B/To G 公司
- `[核心客户]`：前 5/10 大客户
- `[采购方类型]`：政府 / 国企 / 军队 / 事业单位 / 大型民企
- `[平台]`：ccgp / ctbpsp / ggzy / 寻标宝 / 标事通 / 标找找
- `[公告类型]`：采购意向 / 招标 / 中标候选人 / 中标 / 合同 / 废标
- `[中标金额]`：单笔或累计
- `[收入覆盖比]`：招投标可验证收入 / 宣称收入
