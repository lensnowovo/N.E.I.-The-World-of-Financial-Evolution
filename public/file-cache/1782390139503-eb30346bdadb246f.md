# 商业航天投资研究框架

## 这是什么
商业航天（火箭发射/卫星制造/卫星互联网/卫星应用）的投资分析框架。这是 PE/VC 在 2026-2030 年最具"工程经济学"独特性的赛道——**发射成本曲线 + 运力供需 + 星座组网经济性**构成一套无现成框架可套的判断逻辑。适合硬科技 PE/VC、产业资本、国家队配套基金做赛道判断与尽调。核心矛盾：**需求端（万颗星座）确定性极高，供给端（发射成本、批量产能）是瓶颈，产业链利润随瓶颈点迁移。**

## 行业研究框架（6 维度）

### 维度 1：核心投资逻辑——需求扩张 × 供给变革双向共振

**需求端（高度确定性）**：
- 中国星网（GW）+ 千帆（G60）两大万颗级低轨星座规划——未来 5-10 年发射数万颗卫星。
- 新应用：手机直连卫星、天地一体化通信、太空算力。
- 政策目标：**2027 年建成完善体系 + 2030 年国内市场突破 5000 亿元 + 2035 年破万亿**。

**供给端（关键变量）**：
- **可回收火箭**：使发射成本数量级下降（万元/公斤 → 千元/公斤），打开高频发射可能。
- **卫星标准化/批量化**：从定制走向流水线（SpaceX 日产 6 颗 Starlink）。

**共振效应**：基础设施扩张周期，产业链数年高景气。

### 维度 2：发射成本曲线（最核心量化指标）

| 技术阶段 | 单位发射成本 | 代表 | 含义 |
|---|---|---|---|
| 传统一次性火箭 | 1-2 万美元/公斤 | 早期全球商业发射 | 高门槛，仅国家工程 |
| SpaceX 猎鹰 9 号（可回收） | ~3000 美元/公斤 | 当前全球标杆（占 80%+ 商业发射） | 数量级下降 |
| 未来完全复用 | 目标 < 1000 美元/公斤 | 星舰/朱雀可回收路线 | 打开 To C 场景 |

**投资判断**：发射成本下降是整个行业的"摩尔定律"——成本每下降一级，打开一个数量级的商业场景。

### 维度 3：产业链结构与受益顺序（关键）

```
上游（材料/元器件/零部件）   ← 先受益，确定性最高
├── 宇航级高分子材料
├── 高可靠芯片（抗辐射）
├── 3D 打印工艺
├── 发动机（液体/固体）
└── 测控/电气系统

中游（火箭制造 + 卫星制造）  ← 中期兑现
├── 火箭整机（国家队/民营双雄）
├── 卫星平台化制造
└── 发射服务

下游（卫星运营 + 应用）      ← 长期价值，3-5 年后
├── 卫星互联网运营（垄断优势）
├── 遥感数据服务
├── 导航增强
└── To C 应用（手机直连）
```

**受益时序判断**：
| 阶段 | 时间窗 | 受益环节 | 业绩确定性 |
|---|---|---|---|
| **短期（1-2 年）** | 2026-2027 | 上游零部件 + 发动机 + 发射服务 | ★★★★★ |
| **中期（2-4 年）** | 2027-2029 | 卫星批量化制造 | ★★★★ |
| **长期（4-7 年）** | 2029-2032 | 卫星应用 + 运营服务 | ★★★ |

### 维度 4：五大壁垒（极高进入门槛）

1. **系统工程壁垒**：火箭/卫星是多学科交叉系统，任一环节短板即任务失败。
2. **资本壁垒**：火箭研发需数十亿初期投资；一次发射失败 = 数亿元损失。
3. **客户认证壁垒**：航天产品需 "flight heritage"（飞行履历），进入国家级星座供应链需严格资质审核。
4. **规模壁垒**：高频发射/批量生产快速摊薄成本，后入者无订单支撑难降本。
5. **政策/资质壁垒**：发射许可、频谱使用许可、武器装备科研生产许可——稀缺牌照。

### 维度 5：商业模式与盈利路径

**收入多元化矩阵**：
- 制造合同（To G/To B）：火箭、卫星、地面设备。
- 发射服务费：按公斤/按发计费。
- 运营订阅：数据即服务（DaaS）、通信带宽订阅。
- 应用层：农业遥感、海洋监测、物联网连接。

**商业化周期判断**：
- **从技术可行到真正赚钱需 3-5 年甚至更久**——终端普及、价格下探、监管支持三者缺一不可。
- To C 卫星应用不会在 3-5 年内大规模商业化——付费习惯、终端成本、频谱分配都是制约。

### 维度 6：中国特色——"国家队主导 + 民营双驱"

| 阵营 | 代表 | 优势 | 劣势 |
|---|---|---|---|
| **国家队** | 航天科技/科工集团、星网、千帆 | 完整产业链 + 政策资源 + 国家工程 | 商业化效率偏低 |
| **民营火箭双雄** | 蓝箭航天（朱雀/液氧甲烷/已 IPO 申请）、天兵科技 | 技术路线对标 SpaceX，迭代快 | 资金压力大，订单依赖 |
| **民营卫星/运营** | 银河航天、垣信卫星 | 批量化生产 + 应用探索 | 牌照/频谱资源依赖 |

**全球格局**：SpaceX 是"一超"（占 80%+ 全球商业发射），中国是"多强"中最具追赶潜力的。

## 行业核心指标

| 指标 | 健康阈值 | 说明 |
|---|---|---|
| **单位发射成本（美元/公斤）** | < 5000 | 关键成本曲线 |
| **发射成功率** | > 95% | 航天级可靠性 |
| **年发射次数** | 趋势上升 | 运力规模 |
| **可回收次数（单枚火箭）** | > 10 次 | 复用经济性门槛 |
| **卫星年产能** | > 100 颗/年（批量化） | Starlink 标准：6 颗/天 |
| **在手订单 / 年营收** | > 2 倍 | 订单能见度 |
| **国家级星座供应商资质数** | ≥ 1 | 准入门槛 |
| **To G 收入占比** | 40-70% | 基础但需向 To B/C 扩展 |
| **研发费用率** | > 15% | 技术驱动 |
| **现金 runway** | > 24 个月 | 重资本行业生命线 |

## 壁垒判断

### 真壁垒（稀缺且难复制）
1. **可回收火箭飞控算法 + 发动机多次点火**：10 年+ 技术积累 + 海量试验数据。
2. **国家级星座供应商资质**：稀缺牌照，准入门槛。
3. **卫星批量化产线（超级工厂）**：规模效应 + 先发优势。
4. **频谱资源**：低轨频谱"先到先得"——国际协调规则下，先发优势显著。
5. **flight heritage（成功飞行履历）**：客户信任基础，无履历即无订单。

### 假壁垒（警惕）
1. **"已完成发射"≠ 可回收复用**：一次性发射 vs 可回收差一个数量级。
2. **技术先进但无订单**：无飞行履历 = 无客户。
3. **估值靠"对标 SpaceX"叙事**：SpaceX 是 20 年 + 百亿美元投入的结果。

## 红旗识别
1. **发射失败率 > 10% 且无改善**——技术不成熟或质量控制失灵。
2. **单一客户依赖 > 70%**（如仅靠国家队订单）——议价权弱，回款慢。
3. **现金 runway < 18 个月 + 无 IPO 路径**——重资本行业致命。
4. **宣称"对标 SpaceX"但发射成功率 < 80%**——差距非数量级而是代际。
5. **卫星制造成本未随规模下降**——批量产能未真正落地。
6. **核心团队流失（尤其总师级）**——系统工程知识沉淀在团队。
7. **频谱/牌照申请无进展**——准入资质风险。
8. **To C 应用收入占比宣称 > 20%**（当前阶段）——3-5 年内不可能，纯叙事。
9. **重资产投产但订单不足（产能利用率 < 40%）**——固定成本拖累。
10. **管理层用"国家战略"回避商业化问题**——无盈利路径。
