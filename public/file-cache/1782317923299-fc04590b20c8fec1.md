# 财报分析：从业绩会 + 财报快速出点评

这是 Anthropic 官方的 财报分析 skill。公司发财报后快速出一份点评：

- 核心指标：收入 / 利润 / 指引，对比预期和去年同期

- 分业务拆解：各板块增速、毛利率变化

- 电话会要点：管理层表态、前瞻信号

- 模型更新 + 关键问题清单

适合：跟踪持仓或目标公司财报、写点评的人。二级市场研究员和一级市场投后都用得上。
