# 先进制造 / 机器人投资研究框架

## 这是什么
先进制造（数控机床/激光/机器人/汽车零部件/特种材料）的投资分析框架，特别聚焦 **人形机器人** 与 **AI+制造** 这一波产业跃迁。适合 **PE/VC/产业资本** 在硬科技/国产替代主题下做赛道扫描、立项尽调。核心判断逻辑：**产业链分工中，核心零部件（上游）壁垒最高、利润最厚；中游整机拼设计与算法集成；下游拼场景落地。**

## 行业研究框架（6 维度）

### 维度 1：产业链结构（先画图，再判断价值分配）

```
上游（核心零部件 + 软件）        ← 壁垒最高，利润最厚
├── 减速器（谐波/行星滚柱丝杠）
├── 伺服系统/电机（无框力矩电机）
├── 控制器
├── 传感器（六维力/视觉）
├── 热管理
├── 芯片
└── 大模型/控制算法（机器人的"灵魂"）

中游（本体制造 + 系统集成）      ← 拼设计 + 算法集成 + 规模化生产
├── 传统机器人制造商（四大家族）
├── 新兴人形机器人厂家（Optimus/宇树/智元）
├── 互联网巨头（切入具身智能）
└── 智驾厂商（技术复用）

下游（多元应用场景）             ← 拼场景理解 + 客户关系
├── 工业场景（物流分拣/装配/巡检）
├── 医疗康复
└── 家庭服务
```

**投资优先级判断**：上游核心零部件 > 中游整机龙头 > 下游集成商。

### 维度 2：技术壁垒与国产替代进度
- **核心零部件国产化率**：减速器（谐波已国产化较高、RV 减速器日企主导）、伺服电机（国产率 30%+）、控制器、传感器。
- **对标海外竞品的性能差距**：精度、稳定性、寿命——硬件"大而不强"是共识。
- **AI 降低硬件精度要求**：VLM → VLA 模型降低对硬件指标要求，叠加中国制造优势，更适合量产。
- **专利布局**：底层原创 vs 外围改进；全球布局完整性。

### 维度 3：客户结构与定点验证
- **是否进入头部客户供应链**：特斯拉/比亚迪/华为/富士康等。
- **定点验证与批量供货测试**：从样品送样 → 小批量 → 大批量。
- **客户集中度**：单一客户依赖（>50% 营收）vs 多元化。

### 维度 4：汽车产业链复用（关键产业洞察）
- **人形机器人与汽车零部件高度融合**：Optimus 大量复用汽车零部件供应商。
- **投资策略**：寻找"汽车零部件基本盘 + 机器人送样/联合研发"双重属性的企业——基本面稳健 + 成长弹性。
- **国产替代窗口**：智能电动车时代，电机/电控/减速器取代发动机/变速箱——中外企业同一起跑线。

### 维度 5：AI+制造的变量机会
- **数控机床 + AI**：自适应加工、预测性维护、工艺优化。
- **机器人 + 大模型**：从"示教复现"到"自主决策"——场景泛化能力是关键。
- **投资窗口判断**：早期看 EV 供应商协同；成熟期看头部客户的二供/三供。

### 维度 6：中国特色——政策红利与出海
- **政策**："人工智能+"行动、《机器人产业发展规划》、政府引导基金返投方向。
- **出海**：汽车零部件出海"量价齐升"（2024 年出口 641 万辆，单价从 1.4 万→1.83 万美元）。
- **产业链转移**：中国制造的性价比优势 + 国产替代政策红利。

## 三级项目分类框架（高瓴/源码等机构实际使用）

| 级别 | 定义 | 核心要求 | 估值特征 |
|---|---|---|---|
| **学霸项目** | 技术领先的新兴企业 | 技术路线符合全球主流 + 中国性价比优势 + 已通过头部客户验证 | 估值偏高，但技术壁垒高 |
| **牛娃项目** | 二次增长潜力企业 | 客户覆盖行业头部 + 在"人机料法环"5 环节中 ≥3 个有优势 | 估值偏高，但增长快 |
| **普娃项目** | 成熟期企业 | 低估值 + 一二级价差 + 经营确定 + 10%+ 增速 | 估值低，稳健 |

## 行业核心指标

| 指标 | 判断标准 | 说明 |
|---|---|---|
| **核心零部件国产化率** | < 30% = 替代空间大；> 70% = 已成熟 | 决定投资机会窗口 |
| **良率** | > 95% = 优秀；< 85% = 量产风险 | 折叠屏铰链/精密件 |
| **产能利用率** | > 70% = 健康；< 60% = 产能过剩 | |
| **头部客户渗透率** | 进入 Top 3 客户供应链 = 强信号 | |
| **研发费用率** | > 8% = 技术驱动；< 3% = 成本驱动 | |
| **毛利率** | 核心零部件 > 40%；整机 20-30%；集成 < 20% | 产业链位置决定 |
| **送样→小批→大批进度** | 节点明确、时间可控 | |
| **订单能见度** | 在手订单 / 年营收 > 1.5 倍 | |
| **人形机器人单台 BOM 成本** | 当前 ~5-10 万美元，目标 < 2 万美元 | 决定规模化拐点 |

## 壁垒判断

### 真壁垒
1. **精密制造工艺积累**：减速器/丝杠的精度和一致性——10 年+ know-how，非砸钱可破。
2. **客户认证周期**：汽车/机器人零部件认证周期 1-2 年，先入者优势明显。
3. **规模效应**：单位成本随产能指数下降。
4. **材料科学壁垒**：特种功能材料、高分子材料——国产化率低、技术难度高。
5. **AI 算法 + 数据飞轮**：场景数据越多，模型越强，越能降硬件要求。

### 假壁垒（警惕）
1. **"已送样"≠ 已批量**：送样到量产还有 2-3 年。
2. **单一专利数量**：要看底层原创专利占比。
3. **低价中标**：没有技术壁垒的低价 = 产能过剩前兆。

## 红旗识别
1. **核心零部件单一海外供应商，无国产备份方案**——卡脖子风险。
2. **良率低于行业平均且无改善趋势**——量产风险。
3. **客户认证周期长于预期，反复送样无批量订单**——技术不达标。
4. **毛利率持续下滑 + 应收账款激增**——价格战 + 客户质量恶化。
5. **研发费用率 < 3% 却宣称"技术领先"**——名不副实。
6. **产能利用率 < 60% 仍在扩产**——故事驱动而非需求驱动。
7. **核心团队来自海外竞品但无知识产权切割**——侵权风险。
8. **AI 叙事强但无实际场景落地数据**——纯概念。
9. **严重依赖政府补贴维持盈利**——商业模式未跑通。
10. **汽车业务基本盘萎缩而机器人业务尚未放量**——青黄不接。
