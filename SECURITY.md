# Security Policy

请通过 GitHub Private Vulnerability Reporting 私下报告安全问题：

https://github.com/lensnowovo/N.E.I.-The-World-of-Financial-Evolution/security/advisories/new

请提供受影响 URL/版本、复现步骤、影响范围和建议修复。不要在公开 Issue、群聊或截图中发送数据库连接串、API Key、完整 MCP Token、用户个人信息或项目材料。

我们会尽快确认报告、限制风险扩散并协调修复。涉及个人信息或网络安全事件时，将依适用法律履行补救、报告和通知义务。
