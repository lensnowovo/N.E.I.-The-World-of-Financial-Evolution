# N.E.I. 中国境内网站合规与安全基线

版本：2026-07-22  
适用范围：`nei-pevc.com`、N.E.I. MCP、网站端 Memory Node 授权、投稿与评论功能。

本文件是产品与工程执行清单，不替代执业律师、网安部门或备案主管机关的正式意见。规则是否触发取决于实际主体、用户规模、收费模式、数据类型和部署区域。

## 1. 当前业务边界

- 公开浏览：Skill / Workflow / MCP 目录。
- 账号：邮箱注册、GitHub 登录、个人资料。
- 互动：投稿、附件、评论、收藏、关注、举报。
- MCP：Token、全库搜索、收藏管理、Skill 文本分发。
- Memory Node：网站只管理权益、设备和许可证；记忆正文留在用户本机。
- AI：用户主动使用的转写/补全，以及公开投稿的安全检查。

## 3. 主要法规与适用点

### 3.1 备案和网络安全

- 《非经营性互联网信息服务备案管理办法》第十三条要求在主页底部中央标明备案编号并链接备案管理系统。  
  https://www.miit.gov.cn/gyhxxhb/jgsj/cyzcyfgs/bmgz/xxtxl/art/2024/art_84a0cfa0ebd049bbbe751dca9a008e56.html
- 《网络安全法》要求网络运营者采取技术措施、保存相关网络日志不少于六个月，并履行事件处置义务。  
  https://www.cac.gov.cn/2016-11/07/c_1119867116_2.htm
- 《国家网络安全事件报告管理办法》自 2025-11-01 施行；达到相应等级的事件需按规定时限报告。  
  https://www.cac.gov.cn/2025-09/15/c_1759583017717009.htm

### 3.2 账号、评论与内容平台

- 《互联网用户账号信息管理规定》第六条要求公开账号管理规则并签订服务协议；第九条规定为用户提供信息发布等服务时，应基于手机号、身份证件或统一社会信用代码等方式进行真实身份认证。  
  https://www.cac.gov.cn/2022-06/26/c_1657868775042841.htm
- 《互联网跟帖评论服务管理规定》要求落实真实身份认证、先审后发、实时巡查、举报和处置记录。  
  https://www.cac.gov.cn/2022-11/16/c_1670253725725039.htm
- 《网络信息内容生态治理规定》要求内容服务平台承担信息内容管理主体责任。  
  https://www.cac.gov.cn/2019-12/20/c_1578375159509309.htm

### 3.3 个人信息

- 《个人信息保护法》要求明确告知处理者、目的、方式、种类、保存期限和权利行使方式；向其他处理者提供、处理敏感个人信息、跨境提供等场景可能需要单独同意和影响评估。  
  https://www.miit.gov.cn/jgsj/zfs/fl/art/2022/art_515a4b20c12f430eab54bb4f56d89f56.html
- 《网络数据安全管理条例》进一步明确网络数据处理者的制度、事件和跨境义务。  
  https://www.cac.gov.cn/2024-09/30/c_1729384452307680.htm
- 《个人信息保护合规审计管理办法》及其指引自 2025-05-01 施行，要求审查同意、最短保存、个人权利、内部制度和事件响应。  
  https://www.cac.gov.cn/2025-02/14/c_1741233507681519.htm

### 3.4 AI 内容与投资服务

- 《人工智能生成合成内容标识办法》自 2025-09-01 施行；平台若提供生成或传播 AI 生成内容，应评估显式/隐式标识和用户声明义务。  
  https://www.cac.gov.cn/2025-03/14/c_1743654684782215.htm
- N.E.I. 当前应坚持方法库和研究辅助定位，避免具体证券买卖、时点、收益保证或代客决策等未经许可的投资咨询表达。证监系统风险提示可作为运营红线参考。  
  https://www.csrc.gov.cn/beijing/c105537/c7643305/content.shtml

## 4. 产品决策门

以下项目不能用前端文案替代，需要负责人明确选择：

1. **投稿与评论实名**：当前采用“普通用户投稿统一待审、暂停公开评论/评论点赞/关注”；未来若恢复社交互动，先接手机实名、审核、举报和处置日志。
2. **商业化**：收费前确定经营主体、税务、退款、自动续费、电子合同、支付渠道和是否触发经营性互联网/增值电信许可评估。
3. **跨境架构**：继续使用境外数据库/托管并履行告知、单独同意、影响评估；或逐步迁移中国境内基础设施。
4. **Memory Node 内测**：授权 refresh、设备查看/撤销、客户端下载完整前，不公开发放。

## 5. 每月自查

- 核对页脚备案号、公安备案号和链接。
- 抽查 10 个新增投稿的来源、许可证、AI 声明和敏感信息。
- 复核管理员、对象存储、数据库、Vercel、邮件和 GitHub 权限。
- 检查 MCP Token 异常、审核失败、举报积压、错误率和日志清理。
- 更新第三方处理者、数据区域、协议版本和联系渠道。
- 验证数据导出、注销、版权下架和安全事件演练。
