# 数据清单与保留基线

| 数据 | 目的 | 主要存储/处理方 | 对外提供 | 基线保留 |
| --- | --- | --- | --- | --- |
| 邮箱、昵称、角色、资料 | 注册与个人主页 | Neon | Vercel 运行时；Resend 获得邮箱 | 账号存续期 |
| 密码哈希、会话 | 认证 | Neon / Cookie | Vercel | 会话最长 30 天；哈希至账号删除 |
| 验证码记录 | 注册/登录/重置 | Neon / Resend | Resend 获得邮箱与邮件 | 30 天内清理 |
| IP、User-Agent、限流桶 | 安全、同意证据 | Neon / Vercel | Vercel | 按用途最短保存；限流过期即清 |
| 投稿、评论、举报、收藏 | 社区服务与审核 | Neon | 公开内容向访问者/MCP | 删除、下架或争议结束前 |
| 附件 | Skill 资料分发 | 配置的 S3 兼容存储（当前应核对阿里云 OSS） | 公开帖子附件向访问者 | 帖子删除后按策略清理 |
| MCP Token 哈希 | Agent 鉴权 | Neon | 无明文外发 | 撤销/账号删除后清理 |
| MCP 调用元数据 | 安全、状态、运营 | Neon | 无 | 180 天 |
| Memory 权益/设备/许可证 | 激活、设备上限、订阅 | Neon | Memory Node 客户端 | 权益期及安全/争议所需期间 |
| Memory 记忆正文 | 本地记忆服务 | 用户本机 | 默认不上传 N.E.I. | 由用户本机控制 |
| AI 辅助文本 | 用户主动转写/补全、公开投稿扫描 | 智谱 GLM | 智谱 | 以供应商条款和最短必要为准 |

上线前必须由负责人逐项核对生产环境，不允许政策写 R2 而实际使用 OSS，或政策写国内而实际部署境外。
