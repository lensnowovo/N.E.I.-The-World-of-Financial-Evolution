# N.E.I. Public Beta 上线检查清单

这份清单用于 Public Beta 前最后一轮人工验收。目标不是继续堆功能，而是确认 N.E.I. 已经能被理解、能被使用、能被信任、能被分享。

## 1. 工程与部署

- [ ] `npm install` 可正常完成
- [ ] `npm run lint` 通过
- [ ] `npx tsc --noEmit` 通过
- [ ] `npm run build` 通过
- [ ] Vercel 环境变量完整
- [ ] `NEXT_PUBLIC_BASE_URL` 是正式线上域名
- [ ] `SESSION_SECRET` 足够随机且只存在于服务端环境变量
- [ ] Prisma Client 在 Vercel build 阶段正常生成
- [ ] Neon 数据库 schema 与 `prisma/schema.prisma` 一致
- [ ] 数据库连接池适合 serverless 环境
- [ ] OSS / S3 上传、下载在生产环境可用

## 2. 账号与身份

- [ ] 邮箱注册流程可完成
- [ ] GitHub OAuth 可登录
- [ ] 登录后可编辑昵称、头像、身份、机构、简介
- [ ] 用户主页能展示发布记录、收藏记录、关注/粉丝数
- [ ] 新增身份可正常保存：产业投资、政府引导基金、投后、LP、研究员
- [ ] 贡献者徽章展示合理，不出现虚假承诺

## 3. Skill 发现与使用

- [ ] 首页 10 秒内能看懂：N.E.I. 是一级市场投资人的 AI Skill Hub
- [ ] 8 个任务入口可点击，并能进入合理筛选结果
- [ ] 搜索、场景筛选、类型筛选、行业筛选可组合使用
- [ ] 空状态文案清楚，引导用户放宽筛选或发布 Skill
- [ ] Skill 详情页可打开，无生产 500
- [ ] 详情页复制、收藏、下载附件可用
- [ ] 未登录用户点击收藏/执行时能跳转登录并回跳

## 4. MCP

- [ ] 用户可生成 MCP Token
- [ ] 重新生成 Token 后旧 Token 不再可用
- [ ] `/mcp` 指南里的 server URL 正确
- [ ] MCP `search_skills` 可返回公开 Skill
- [ ] MCP `get_skill` 可返回指定 Skill 文本
- [ ] MCP `list_my_skills` 只返回当前 Token 用户收藏的 Skill
- [ ] 页面明确说明：MCP 不读取本地文件、不上传 BP/财务模型/投委会材料

## 5. 内容质量与信任

- [ ] 首页精选区域只出现结构完整、可复用内容
- [ ] 官方内容、社区贡献、外部转载来源标识清楚
- [ ] MCP Ready 内容正文足够完整，能被 AI 客户端直接理解
- [ ] 每个核心任务场景至少有 2-3 个可用 Skill
- [ ] `/legal` 页面可访问
- [ ] Footer 有信任规则、MCP 安全说明和连接配置入口

## 6. 分享与移动端

- [ ] 首页移动端无横向滚动和明显重叠
- [ ] 列表页移动端可筛选、可浏览
- [ ] 详情页移动端可复制、收藏、下载
- [ ] 详情页 Open Graph 标题和描述正确
- [ ] 微信 / 飞书 / Slack 分享链接时能看懂标题和摘要
- [ ] 404 和 error page 可用

## 7. 人工复核

- [ ] 不把 N.E.I. 描述成投资建议平台
- [ ] 不承诺 AI 输出一定准确
- [ ] 不诱导用户上传未脱敏项目材料
- [ ] 不把 Beta 阶段轻量条款当正式法律文件
- [ ] 首页、详情页、MCP 页面没有明显机器味或泛 AI SaaS 味
