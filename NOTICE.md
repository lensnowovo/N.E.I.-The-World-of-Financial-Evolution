# N.E.I. / nei-pevc.com Notice

## Canonical repository

This repository, `lensnowovo/N.E.I.-The-World-of-Financial-Evolution`, is the canonical product repository for `nei-pevc.com`.

`nei-pevc.com` is operated as a PEVC-focused product line under the broader N.E.I. community context. Its product direction, deployment, infrastructure, domain, security system, MCP workflow, content model, admin operations, and ongoing development are led, funded, and maintained by lensnowovo.

## Project origin

N.E.I. began as a community idea around AI workflows for private-market professionals. An early MVP was scaffolded with support from jammyssy using Claude Code. That initial scaffold served as a starting point for later product development.

The current `nei-pevc.com` codebase has since evolved substantially beyond the original MVP, including the public site, MCP onboarding and server capabilities, security and trust pages, Skill quality system, admin operations, deployment configuration, smoke tests, and product launch documentation.
